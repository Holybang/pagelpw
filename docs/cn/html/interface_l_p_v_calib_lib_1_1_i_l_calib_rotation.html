详细描述

该接口提供旋转中心标定和对位的功能。

旋转中心标定通常用于带有旋转平台的机器视觉系统，广泛应用于引导、抓取、对位等视觉任务。
基于围绕着该中心点旋转的一个或多个目标点的轨迹，优化得到其旋转中心。 旋转中心标定通常在手眼标定（使用多点变换标定模型）后进行。 因此，输入的点位置为其物理坐标位置。

标定后，可基于旋转中心，对齐目标点与对象点，计算将对象点齐平目标点所需的平移量和旋转角度。

使用这个接口，需要创建一个 LCalibRotation 对象。

示例代码

C++
Code snippets: snippets/CalibRotationCenter.cpp
ILCalibRotationPtr m_rot_calib(__uuidof(LCalibRotation));
m_rot_calib->AddRotPointPair(0, 100, 0, -100, 0);
m_rot_calib->AddRotPointPair(100, 100, -100, -100, 45);
m_rot_calib->AddRotPointPair(100, 0, -100, 0, 90);
LPVErrorCode err = m_rot_calib->Calibrate(); // the rotation center should be (0, 0)
if (err != LPVErrorCode::LPVNoError || !m_rot_calib->IsCalibrated()) return;
m_rot_calib->SetAim(0, 10, 0, 90);
double dx, dy, da;
m_rot_calib->Align(10, 0, 90, 0, &dx, &dy, &da);
