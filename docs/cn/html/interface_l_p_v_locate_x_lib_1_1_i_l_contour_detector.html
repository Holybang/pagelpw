详细描述

该接口提供轮廓定位的功能。

轮廓定位用于在输入图像中和指定的区域中，定位一条或多条轮廓。
对于图像中的显著边缘像素，该算法基于方向一致性连接相邻边缘，形成连续的轮廓。 支持基于轮廓的强度和长度进一步过滤较短、较弱的轮廓，或基于区域指定的扫描方向，按照梯度方向过滤轮廓。
该算法具有高实时性、高亚像素精度等特点。

使用这个接口，需要创建一个 LContourDetector 对象。

示例代码

C++
ILContourDetectorPtr m_contour_detector(__uuidof(LContourDetector));
m_contour_detector->ConnectTolerance = 16;
m_contour_detector->KSize = 5;
m_contour_detector->RejectShort = 100;
LPVErrorCode err = m_contour_detector->DetectWithDir(m_img, m_rot_roi,
LPVPolarity::LPVBlack2White, 30, 70, &m_contour_results);
if (err != LPVErrorCode::LPVNoError) return;

检测和定位轮廓，并按轮廓走向和边缘转换方向过滤。
对于轮廓中的每个像素，计算其边缘转换方向与区域扫描方向间的差异。差异过大的像素将被从轮廓中剔除。 若轮廓中存在过多的剔除像素且占比大于设定阈值，则认为轮廓的主要方向与所需不符，该其从最终结果中移除。

参数
[in]img 输入的图像
[in]region输入的区域，一个旋转矩形区域对象，以其旋转角度为扫描方向。
[in]polarity边缘极性，相对于区域的扫描方向
[in]tolerance方向差异的角度容差
[in]threshold方向不一致像素的最大占比百分数。超过这一数值的轮廓被认为是方向不一致的轮廓。
[out]results返回轮廓定位结果
返回值
error返回错误码
