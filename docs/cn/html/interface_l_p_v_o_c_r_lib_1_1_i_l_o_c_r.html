详细描述

该接口类提供字符分割、识别和质量检测功能。

OCR 工具用于基于视觉技术的字符分割、识别和质量检测。 支持一次提取和分割多行文本，并能够分割激光打印标签中常见的行内连词和行间粘连。 字符识别和质量检测基于关联的字库及其训练的字符模型进行。

使用这个接口，需要创建一个 LOCR 对象。

示例代码

C++
ILOCRPtr ocr_tool(__uuidof(LOCR));
ocr_tool->FontDatabase = font_db;
ocr_tool->SetCharSize(100, 150, 20, 20); // 100 x 150 +- 20%
ocr_tool->NoiseThreshold = 20;
ILOCRResultPtr ocr_result;
LPVErrorCode err = ocr_tool->Detect(img, roi, &ocr_result);
if (err != LPVErrorCode::LPVNoError) return; // failed, or no result
