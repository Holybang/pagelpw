详细描述

该接口对应单个晶圆图，用于提供晶圆图生成、存储、读取、显示绘制，以及 DIE 访问和数量统计等功能。

晶圆图是集成电路生产中用于标识晶圆中芯片位置和质量的一种重要手段，已在封测行业广泛替代晶圆上墨水。 LWaferMap 提供了多样的对晶圆图的编辑、浏览、统计等功能，以满足面向半导体和 IC 行业的晶圆图显示、更新、输出等需求。

使用这个接口，需要创建一个 LWaferMap 对象。

示例代码

C++
ILWaferMapPtr templateMap(__uuidof(LWaferMap));
templateMap->Load(L"template.sinf");
// move the center of the template map to (0, 0)
templateMap->WaferProps->CenterX = 0;
templateMap->WaferProps->CenterY = 0;
ILWaferSorterPtr sorter(__uuidof(LWaferSorter));
// set the template
sorter->Template = templateMap;
// add dice
// sort
ILWaferMapPtr wmap;
LPVErrorCode err = sorter->Sort(&wmap);
if (err != LPVErrorCode.LPVNoError) return;
// count the pass dice
int passCount = wmap->CountByQuality(LPWBinPass);
int totalCount = wmap->Count();
double passPercentage = (double)passCount / totalCount;

LWaferMap templateMap = new LWaferMap();
templateMap.Load(L"template.sinf");
// move the center of the template map to (0, 0)
templateMap.WaferProps.CenterX = 0;
templateMap.WaferProps.CenterY = 0;
LWaferSorter sorter = new LWaferSorter();

sorter.Template = templateMap;
// add dice
// sort

◆ Synthesize()

LPVErrorCode Synthesize ( ILWaferProps *  prop)

基于给定的属性配置，生成晶圆图。
在生成过程中，所有完全落在晶圆外部的位置将被标记为 Null，所有完整地位于晶圆以及边距内部的位置将被标记为 Pass， 其余与边距相交或边距外的位置则为 Edge。
生成完成后，可以通过 Die 或者 DieQuality 修改/增加其他类型的 DIE。

参数
[in]prop输入的晶圆属性配置。必须在指定晶圆的 Step Size 和行列数（或直径）。
返回值
error返回错误码
