玩转 DEMO

我们提供两种类型的 DEMO：

  • 带有 GUI 的 MFC/C#/QT 程序，提供可视化界面用于参数的调整、结果显示等。其可执行程序位于 demo 文件夹下的所需平台对应的文件夹内，如 demo/x64 ，并提供相应的源代码和工程文件。
  • 简单的 C++/C# 代码片段，用于演示某些功能函数如何调用。通常为一个 Main 函数体，位于 demo/snippets 文件夹内。使用方法请参考 如何使用 DEMO 代码片段。

注意 1：DEMO 程序的平台需与安装注册的 LPW 版本对应。
注意 2：执行 DEMO 程序前，需保证 LPW 已正常注册和授权。
注意 3：安装后自带的 Qt DEMO 程序基于 Qt 5.9.4 版本编译，运行这些 Qt DEMO 的可执行文件需要从 Qt 官方网站下载并安装对应版本。或者，使用我们提供的只包含依赖项的精简压缩包，下载后（ x86 或 x64 ）解压到对应的 DEMO 可执行文件的同级目录下。

基础功能

演示典型的视觉检测任务流程，包括图像、Region、模板、标定等基础功能。

源代码 (C++)： mfcdemo/MFCSanity
源代码 (Qt)： qtdemo/QtSanity
源代码 (C#)： csharpdemo/CSharpSanity
图像文件： sourceImg.png，templateImg.png

相关类： LImage, LRectRegion, LMatch, LMatchResults, LCalibNPoints

  1. 点击 Load Image 按钮，加载图片。
  2. 点击 Template Learn 按钮，按照 Region 的位置抠图，学习模板。
  3. 点击 Template Match 按钮，执行模板匹配，右侧显示像素坐标。
  4. 点击 Calibrate 按钮，执行9点标定，右侧显示物理坐标。
  5. 点击 Reset Calibration 按钮，重置标定，右侧显示像素坐标。
  6. 点击 Save Calibration 按钮，保存标定文件，右侧显示物理坐标。
  7. 点击 Load Calibration 按钮，加载标定文件。
  8. 点击 Copy & Save 按钮，执行保存图像，在可执行文件同级的 images 目录保存拷贝的 jpg 图像。
  9. 拖动图形界面中的 Region 边界，可移动或改变 Region 形状。
  10. 点击 Reset Roi 按钮，按照当前图像尺寸重置roi位置。


Region 操作

演示如何进行编辑 Region 及如何计算生成新的 Region。

源代码 (C++)： mfcdemo/MFCRegion
源代码 (Qt)： qtdemo/QtRegion
源代码 (C#)： csharpdemo/CSharpRegion
图像文件： mask.bmp

相关类： LRectRegion, LRotRectRegion, LPolyRegion, LMaskRegion, LCircleRegion, LAnnulusRegion

  1. 点击 Load Image 按钮，加载图像。
  2. 点击 Load Mask 按钮，加载蒙版图像到 Mask Region。
  3. 选择 R1 和 R2 用于后续的操作。
  4. 点击下方的各类操作按钮，运算得到的新的 Region 显示在画布上。
  5. 该演示程序可对选择的 R1 和 R2 执行 Union， Subtract，Intersect ，或者 Exclusive OR 操作。
  6. 该演示程序可对选择的 R1 执行 Translation，Rotate，Scale 或者 Invert 操作，使用输入的 dx，dy，da 和 ds 数值。
  7. 点击 Reduce Image 按钮，计算和显示 R1 的子图像和对应的蒙版。
  8. 点击 To Mask 按钮，显示 R1 的蒙版在原图中的对应位置。
  9. 移动鼠标可显示其当前所在的像素位置，以及在 R1 中的相对位置。
  10. 点击 Back to Edit 按钮回到初始状态。

斑点分析

演示斑点分析，支持灰度和彩色模式。

源代码 (C++)： mfcdemo/MFCBlob
源代码 (Qt)： qtdemo/QtBlob
源代码 (C#)： csharpdemo/CSharpBlob
Image files: rou.jpg, smile.png, cat.png

相关类： LBlobAnalysis, LBlobFilter, LBlobResults, LBlob

  1. 点击 Load Image 按钮，加载图像。
    • Mode 根据加载的图像格式，自动切换为 Grayscale 或 Color 。
  2. 选择合适的 Threshold 类型:
    • 0 Custom: 使用自定义的阈值上下限。
    • 1 Global Adaptive: 使用基于全图的自适应阈值。需设置 Polarity 以及修正阈值时使用的 Bias 数值。
    • 2 Local Adaptive: 使用基于局部区域的的自适应阈值。需设置 Polarity 以及修正阈值时使用的 Bias 数值。
  3. 修改 Soft Threshold 为非零数值开启软阈值功能。
  4. 点击 Preview 按钮，预览阈值处理的中间结果图像。
  5. 或者，可以点击 Select Blob Range - Click on Canvas 按钮，再选择画布中的斑点位置，自动生成斑点对应的阈值范围。
  6. 修改 Max Count 用于多目标检测。
  7. 选择合适的 Sort by 用于对所有候选的斑点进行排序，选择其中靠前的若干个斑点作为最终结果。
  8. 点击 Detect 按钮，执行斑点分析。
  9. 斑点结果显示在下方的表格中，包括每个斑点的中心位置、面积、周长、圆度及长宽比。
  10. 选择其中某一个结果，在画布中高亮显示。
  11. 按需勾选并设置过滤器的上下限。
  12. 点击 Set Limitation (Based on Current Blob) 按钮，基于当前选中的斑点属性，自动设置过滤器的上下限。
  13. 点击 Re-filter 按钮，应用过滤器到斑点结果，得到过滤后的结果列表。

  8. 右侧的表格显示所有检测结果，包括每条直线的中点坐标和角度。

标定

演示相机标定，包括多点、小孔、FFD等相机模型，支持不同类型的标定板。

源代码 (C++)： mfcdemo/MFCCalib
源代码 (Qt)： qtdemo/QtCalib
源代码 (C#)： csharpdemo/CSharpCalib
图像文件： chessboard.png，circlegrid.png，distcharuco.jpg

相关类： LCalibNPoints, LCalibPinHole, LCalibFFD, LCalibCustom, LCalibBoard, LCalibPointPairs

  1. 选择合适的标定板，点击相应按钮：
    • Chessboard ：打开棋盘格图像，设置相应的 Grid Size 和 Square Size 数值。
    • Circle Grid ：打开圆形点阵图像，设置相应的 Grid Size 和 Square Size 数值。
    • Distributed Charuco ：打开分散式二维码棋盘格图像，设置相应的 Square Size 数值。
  2. 勾选 Auto Remove Bad 启用自动剔除检测到的坏点的功能。
  3. 点击 Detect 按钮，检测标定板，生成图像和物理点对集合。
  4. 选择合适的标定模型：
    • NPoints ：多点变换模型，需设置合适的变换矩阵类型。
    • PinHole ：小孔模型，可以点击 Add Multi-view Points ，设置当前的点对集合用于多视角标定。
    • FFD ：FFD模型，需设置合适的网格尺寸，更大的尺寸可获得更高的优化精度，但速度更慢。
    • Custom ：自定义标定模型。
  5. 点击 Calibrate 按钮，执行相机标定。
  6. 点击 Save 按钮，保存当前标定到文件。
  7. 点击 Load 按钮，从文件读取标定。
  8. 点击 Reset 按钮，重置标定。
  9. 选择合适的矫正模式，点击 Fix Image 按钮，执行图像矫正，修正图像的镜头畸变和相机倾斜，并显示结果在画布上。

连接到灵闪服务

演示如何与灵闪服务集成，并使用具有强大功能的灵闪作业。

源代码 (C++)： mfcdemo/MFCIBService
源代码 (Qt)： qtdemo/QtIBService
源代码 (C#)： csharpdemo/CSharpIBService
图像文件夹： exame_chip_1，exame_chip_2

相关类： LIBService, LIBAlgoResult, LIBData, LIBAlgoConfig, LMatch

  1. 点击 Select IB Service 按钮，选择灵闪的安装目录。
  2. 等待该演示程序连接到灵闪服务，成功后弹出窗口提示。
  3. 点击 Load IB Task 按钮，选择样本任务和图像所在的文件夹。
  4. 该演示程序将加载并训练模板，在图像中找到所有匹配目标并显示在右侧列表中。
  5. 在列表中选择其中一个匹配结果。
  6. 该演示程序将基于该结果所在的位置，生成 Region 然后执行灵闪作业。
  7. 执行时间和结果显示在画布和右侧面板中。
  8. 点击 Refresh 按钮，从灵闪服务重新加载作业的参数列表。
  9. 在输入框中修改参数的数值。
  10. 拖动图形界面中的 Region 边缘，可移动或改变作业中 Region 的形状和位置。
  11. 点击 Apply 按钮，应用改动后的参数到灵闪服务，再次执行该作业将看到参数变换的影响。
