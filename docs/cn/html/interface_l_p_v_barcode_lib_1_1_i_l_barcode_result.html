详细描述

该接口提供单个条码检测的结果，可获取其具体属性。

通常通过 ILBarcodeResults::Item() 函数获得 LBarcodeResult 对象。可以使用该接口访问单个条码对象的具体信息，如解码内容、质量评价等， 还可以调用绘制函数将单个条码绘制在界面上。

示例代码

C++
ILBarcodeResult codeResPtr = m_code_results->Item(i);
CString text = codeResPtr->DecodedString;
C#
LBarcodeResult codeResb = m_code_results->Item(i);
String text = codeRes.DecodedString;
