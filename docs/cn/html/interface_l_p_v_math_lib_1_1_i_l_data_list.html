详细描述

该接口类定义了一个浮点型数据队列，可用于度量计算，过滤，分析等任务。

使用这个接口，需要创建一个 LDataList 对象。

示例代码

C++

Code snippets: snippets/ProjectionAndAnalysis.cpp

ILDataListPtr dataList(__uuidof(LDataList));
dataList->Add(1.1);
dataList->Add(2.3);
dataList->Add(4);
double meanValue = dataList->Metric[LPVAggMax];
double majorValue = dataList->Metric[LPVAggMajor];

◆ MaxLocalExtrema()

SAFEARRAY* MaxLocalExtrema ( double  threshold,
int  num,
LPVFindBy  findBy,
BOOL  onePeakMode 
)

查找数据列表中的局部最大极值。

参数
[in]threshold用于过滤数据中噪声的阈值，低于该阈值的数据在局部极值查找时不予考虑。 默认值为最小的 double 值，表示无过滤。
[in]num期望找到的局部极值数量。若存在多于该数量的结果，将按照给定的 findBy 策略，选择评价靠前的若干个结果。 默认设置为 0，表示获取所有局部极值。
[in]findBy搜索策略
[in]onePeakMode是否启用单峰模式。若启用，则在阈值过滤后的同一个区域内，只产生一个最大值的点作为局部极值。默认关闭。
返回值
indexes返回局部极值点对应的索引，升序排列。
参见

◆ MeanFilter()

ILDataList* MeanFilter ( int  kSize)

均值滤波，输出队列中的每个数据表示这一局部区域中所有数据的**均值**，如下所示：

\[ X{i}' = \sum_{j=0}^{kSize} X{i+j-kSize/2} \]

参数
[in]kSize设置滤波工具中使用的核尺寸，需要是一个奇数，如 1, 3, 5, 7 等。默认设置为 3。 尺寸更大的滤波核尺寸通常会带来更强的滤波效果，但速度更慢。
返回值
result输出滤波结果
