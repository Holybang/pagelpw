属性

int MaxCount [get, set]
 最大的图像数量，0 ~ 256。
比如设置为 16 表示将保持 16 张图像的缓存。若超出该数量的图像被添加到该队列中，更早的图像将被丢弃。
默认设置为 0，表示无限制。
 

详细描述

该接口类定义了一个图像队列，用于存储LImage对象。

维护了一个预设长度的缓存池用于存储相机拍摄的多帧图像，后续可用于多帧融合、HDR或图像拼接等任务。

使用这个接口，需要创建一个 LImageList 对象。

示例代码

C++
ILImageListPtr img_list(__uuidof(LImageList));
img_list->MaxCount = 3;
img_list->Add(frame1);
img_list->Add(frame2);
img_list->Add(frame3);
img_list->Add(frame4); // frame 1 is popped
C#
LImageList img_list = new LImageList();
img_list.MaxCount = 3;
img_list.Add(frame1);
img_list.Add(frame2);
img_list.Add(frame3);
img_list.Add(frame4); // frame 1 is popped
