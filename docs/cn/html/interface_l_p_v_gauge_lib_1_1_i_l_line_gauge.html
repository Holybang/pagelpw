详细描述

该接口提供基于卡尺的直线检测功能。

直线卡尺用于检测直线边缘的位置和方向。通过沿着检测区域方向分布的多个边缘卡尺，定位多个有效卡点，最后通过拟合算法获得直线结果。

相比于特征定位模块中的 LLineDetector，直线卡尺可用于检测更不明显的直线，也不容易受到检测区域方向的限制。 但对于有高定位精度要求的场景，仍旧推荐使用 LLineDetector。

使用这个接口，需要创建一个 LLineGauge 对象。

示例代码

C++
ILLineGaugePtr lineGauge(__uuidof(LLineGauge));
lineGauge->Polarity = LPVPolarity::LPVBlack2White;
lineGauge->AcceptScore = 40;
ILGaugeResultPtr lineGaugeResult;
LPVErrorCode err = lineGauge->Detect(img, rot_rect_roi, &lineGaugeResult);
if (err != LPVErrorCode::LPVNoError) return;
