详细描述

LPV 数学与分析库，提供数值计算和分析功能。

该模块用于帮助计算数值的各类度量，并进行数值的过滤、分析和显示。 使用该模块中的类和算法，需按照以下方式引用该模块：

示例代码

C++
#include "LPVMath.h"
C#
using LPVMathLib;
