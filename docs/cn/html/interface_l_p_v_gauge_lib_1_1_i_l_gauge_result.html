详细描述

该接口表示卡尺检测结果，可获取其具体属性。

通常通过各种卡尺的检测函数获得 LGaugeResult 对象，如直线卡尺 LLineGauge 可获得直线形状的检测结果。 可以使用该接口访问检测的具体结果及细节信息，如直线的位置、分数等，还可以调用绘制函数将单个直线对象绘制在界面上。

示例代码

C++
ILLinePtr line = lineGaugeResult->Result;
ILPointsPtr allPoints = lineGaugeResult->SamplePoints;
C#
LLine line = lineGaugeResult.Result;
LPoints allPoints = lineGaugeResult.SamplePoints;

◆ DrawScoreChart()

void DrawScoreChart ( HDC  hdc,
int  index,
LPVChartDrawFlags  drawFlags,
int  xGridStep,
int  yGridStep 
)

绘制检测分数曲线到给定的设备。该曲线表示在沿着检测区域的方向（x 轴）上指定卡点采样区域的检测分数（y 轴）的分布情况。 通常用于为调试合适的分数阈值。

参数
[in]index输入索引，从 0 开始但小于卡点采样区域总数的数字
[in]hdc绘制设备，可为窗口、位图等的上下文句柄。
[in]drawFlags用于控制绘制的内容，查看 LPVChartDrawFlags。
[in]xGridStep绘制 x 轴的步长，默认设置为 0，表示按照分数曲线数据自动选择合适的步长。
[in]yGridStep绘制 y 轴的步长，默认设置为 0，表示按照分数曲线数据自动选择合适的步长。
