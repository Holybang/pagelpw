详细描述

该接口对应单个晶圆，用于保存和获取其设备信息。

使用这个接口，需要创建一个 LWaferProps 对象，或通过 LWaferMap 获取某一个具体位置的 LWaferProps 对象。

示例代码

C++
ILWaferPropsPtr props(__uuidof(LWaferProps));
props->DeviceID = L"My Wafer";
props->Diameter = 200;
props->DieSizeX = 2.25;
props->DieSizeY = 1.35;
// synthesize the wafer map using the properties
ILWaferMapPtr wmap(__uuidof(LWaferMap));
wmap->Synthesize(props);
C#
LWaferProps props = new LWaferProps();
props.DeviceID = "My Wafer";
props.Diameter = 200;
props.DieSizeX = 2.25;
props.DieSizeY = 1.35;
// synthesize the wafer map using the properties
LWaferMap wmap = new LWaferMap();
wmap.Synthesize(props);
