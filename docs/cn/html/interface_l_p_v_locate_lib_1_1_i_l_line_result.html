详细描述

该接口表示单个直线检测结果，可获取其具体属性。

通常通过 ILLineResults::Item() 函数获得 LLineResult 对象。可以使用该接口访问单个直线对象的具体信息，如直线位置、分数等， 还可以调用绘制函数将单个直线对象绘制在界面上。

示例代码

C++
ILLineResultPtr result = m_line_results->Item(i);
ILLinePtr line = result->Line;
C#
LLineResult result = m_line_results.Item(i);
LLine line = result.Line;
