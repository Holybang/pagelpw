详细描述

该接口提供圆定位的功能。

圆定位用于在输入图像和指定的圆环区域中定位一条或多条圆形边缘。
、 该算法具有高实时性、高亚像素精度等特点，其局限性在于其假设定位圆的中心点在指定的圆环区域中心的附近。 因此，使用该算法需要已知圆的大致圆心位置。

使用这个接口，需要创建一个 LCircleDetector 对象。

示例代码

C++
ILCircleDetectorPtr circle_detector(__uuidof(LCircleDetector));
circle_detector->Polarity = LPVPolarity::LPVPolarityEitherEdge;
LPVErrorCode err = circle_detector->Detect(img, roi, &circle_results);
if (err != LPVErrorCode::LPVNoError) return;
