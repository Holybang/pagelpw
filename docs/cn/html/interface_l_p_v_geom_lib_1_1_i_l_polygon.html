◆ ResampleByStep()

ILPolygon* ResampleByStep ( double  step,
double  stepThreshold 
)

沿该多边形采样，按指定距离步长，生成一个新的多边形。 对于非闭合多边形，其端点总是包含在采样结果中。

参数
[in]step采样距离步长

返回值
result返回采样得到的结果多边形
