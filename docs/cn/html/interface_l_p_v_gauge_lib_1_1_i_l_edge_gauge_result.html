详细描述

该接口表示边缘卡尺检测结果，可获取其具体属性。

通常通过边缘卡尺工具 LEdgeGauge 的检测函数获得 LShapeGaugeResult 对象。 可以使用该接口访问检测的具体结果边缘及细节信息，如卡点等，还可以调用绘制函数将该边缘直线及卡点等绘制在界面上。

示例代码

C++
// for double-side edges inspection
ILLinePtr edgeline1, edgeline2;
edgeGaugeResult->GetEdgePair(0, &edgeline1, &edgeline2); // 0 for the first
double w = edgeGaugeResult->GetEdgePairDistance(0);
C#
// for double-side edges inspection
LLine edgeline1, edgeline2;
edgeGaugeResult.GetEdgePair(0, out edgeline1, out edgeline2); // 0 for the first
double w = edgeGaugeResult.GetEdgePairDistance(0);

绘制检测分数曲线到给定的设备。该曲线表示在沿着检测区域的方向（x 轴）上指定卡点采样区域的检测分数（y 轴）的分布情况。 通常用于为调试合适的分数阈值。

参数
[in]hdc绘制设备，可为窗口、位图等的上下文句柄。
[in]drawFlags用于控制绘制的内容，查看 LPVChartDrawFlags。
[in]xGridStep绘制 x 轴的步长，默认设置为 0，表示按照分数曲线数据自动选择合适的步长。
[in]yGridStep绘制 y 轴的步长，默认设置为 0，表示按照分数曲线数据自动选择合适的步长。
