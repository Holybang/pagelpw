详细描述

该接口为图像特征，在机器学习算法中用于从图像中提取特征向量。

首先需通过 Create() 创建指定类型的图像特征，然后通过 ILParameterized 接口调整参数。
下表列出当前支持的所有特征及其参数：

特征参数和使用说明
ORBOriented FAST and Rotated BRIEF
参数名称使用说明类型取值范围默认值
"count"使用 BOW（Bag of Words）模型进行特征表征时的词汇个数整数8 ~ 204864
"patch size"Patch 尺寸整数4 ~ 25632
"scale factor"金字塔的缩放比例浮点数1 ~ 21.2
"scale-down level"金字塔的向下缩放层数整数> 0 5
"random point pair count"ORG 中生成每一项时所使用的点个数整数2 ~ 42
HOGHistogram of Oriented Gradients
参数名称使用说明类型取值范围默认值
"cell size"Cell 尺寸，一个 Cell 用于构建一个直方图尺寸\( 2 \times 2 \) ~ \( 128 \times 128 \)
must be \( 2^x \) 		\( 4 \times 4 \)
"bin size"直方图的统计区域个数，需为 360 或 180（若为无方向梯度）的因子整数1 ~ 609
"cell in block"一个Block 中的 Cell 数量尺寸\( 1 \times 1 \) ~ \( 64 \times 64 \) \( 2 \times 2 \)
"cell in block stride"一个 Block Stride 中的 Cell 数量尺寸\( 1 \times 1 \) ~ \( 64 \times 64 \) \( 1 \times 1 \)
"block stride step"Block Stride 的数量尺寸\( 1 \times 1 \)
"signed gradient"是否使用有方向带符号的梯度，即梯度方向以 360° 或 180° 统计布尔false
AKAZEAccelerated-KAZE
参数名称使用说明类型取值范围默认值
"count"使用 BOW（Bag of Words）模型进行特征表征时的词汇个数整数8 ~ 204864
"rotation invariant"是否旋转无关布尔true
"accept threshold"过滤特征点的阈值浮点数> 0 10
"octave"最大层级数量整数1 ~ 53
"sub-level"每个层级中的子层级数量整数> 0 4
"conductivity"电导系数类型整数查阅 LPVConductivityG2

示例代码

C++
// create a HOG feature
ILFeaturePtr hog(__uuidof(LFeature));
hog->Create(LPVFeatureType::LPVFeatureHOG);
// change its parameters
hog->ParamSize["cell size"] = LSize{ 8, 8 };
hog->ParamI["bin size"] = 5;
hog->ParamSize["cell in block"] = LSize{ 4, 4 };
llc->AddFeature(hog); // add it to the classifier
C#
// create a HOG feature
LFeature hog = new LFeature();
hog.Create(LPVFeatureType.LPVFeatureHOG);
// change its parameters
hog.ParamSize["cell size"] = new LSize{ w = 8, h = 8 };
hog.ParamI["bin size"] = 5;
hog.ParamSize["cell in block"] = new LSize { w = 4, h = 4 };
llc.AddFeature(hog); // add it to the classifier
