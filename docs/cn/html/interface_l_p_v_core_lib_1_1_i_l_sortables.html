详细描述

该接口类用于对一个可排序对象集合进行排序或重整为给定的顺序。

所有提供了排序功能的 LPV 类都实现这个类定义的所有接口。使用这个接口类需将 LPV 的类转换成 ILSortables 接口类。

以下列出了所有实现该接口的可排序集合 LPV 类。

LPVGeomLibLPoints
LPVGeomXLibLSorter
LPVPatLibLMatchResults
LPVBlobLibLBlobResults
LPVLocateLibLLineResults, LCircleResults
LPVLocateXLibLPathResults, LContourResults
LPVBarcodeLibLBarcodeResults

示例代码

C++
((ILSortablesPtr)points)->Sort(LPVSortBy::LPVSortArrayByX,
LPVSortOrder::LPVSortOrderAscending, LPVPositionMode::LPVPosModeCenter);
C#
((ILSortables)points).Sort(LPVSortBy.LPVSortArrayByX,
LPVSortOrder.LPVSortOrderAscending, LPVPositionMode.LPVPosModeCenter);
