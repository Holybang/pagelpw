详细描述

该接口对应一个环状扇形区域对象。

使用这个接口，需要创建一个 LAnnulusSectorRegion 对象。

示例代码

C++
ILAnnulusSectorRegionPtr annulus_sector_roi(__uuid(LAnnulusSectorRegion));
annulus_sector_roi->SetPlacement(300, 300, 100, 200, -30, 15);
C#
LAnnulusSectorRegion annulus_sector_roi = new LAnnulusSectorRegion();
annulus_sector_roi.SetPlacement(300, 300, 100, 200, -30, 15);
