ILWaferMap Template [get, set]
 模板晶圆图。实时晶圆图和最终排序后的晶圆图将对齐到该模板，基于对应的参考 DIE 的位置变换以及晶圆整体的分布情况。
 

详细描述

该接口提供晶圆图排序的功能，用于生成实时或最终排序后的晶圆图。

晶圆图排序需要指定晶圆图模板，实时地加入到排序类的 DIE 放置到与晶圆模板最相近的阵列位置。 在最终排序中，全局优化所有 DIE 的阵列位置，过滤重复位置的 DIE 和噪声，生成最终的晶圆图。

使用这个接口，需要创建一个 LWaferSorter 对象。

示例代码

C++
ILWaferSorterPtr sorter(__uuidof(LWaferSorter));
sorter->Template = templateMap;
// capture and add the reference dice, it helps to align the wafer map to the template
for (int i = 0; i < count; ++i) {
sorter->AddDie2(x, y, LPWBinRef, 0);
}
// the nth capture
for (int i = 0; i < count; ++i) {
sorter->AddDie2(x, y, LPWBinPass, 0);
}
// get the realtime wafer map
ILWaferMapPtr realtimeMap;
LPVErrorCode err = sorter->Realtime(&realtimeMap);
if (err != LPVErrorCode::LPVNoError) return;
// finished, get the final sorted wafer map
ILWaferMapPtr sortedMap;
err = sorter->Sort(&sortedMap);
if (err != LPVErrorCode::LPVNoError) return;

◆ Realtime()

LPVErrorCode Realtime ( ILWaferMap **  realtimeWaferMap)

生成实时晶圆图。通常在晶圆扫描过程中进行，用于高效地生成部分填充的晶圆图，展示当前晶圆的扫描状况。
所有已知的 DIE 基于模板晶圆图上的最近位置进行摆放。
若已有参考 DIE，实时晶圆图将基于对应的参考 DIE 的位置变换，对齐到模板晶圆图。 因此，推荐在晶圆扫描前，先进行参考 DIE 的识别并将其加入排序。
生成实时晶圆图的速度快于最终排序，但同时可能输出错误的 BIN（缺失的或者多余的）。

参数
[in]realtimeWaferMap输出实时晶圆图
返回值
error返回错误码

◆ Sort()

LPVErrorCode Sort ( ILWaferMap **  sortedWaferMap)

排序并生成排序晶圆图。通常在晶圆扫描结束后进行，整个过程耗时较实时晶圆图更长。 所有 DIE 将按照其与邻近 DIE 的相对位置，全局优化后组成整列。重复和错位的位置将被作为噪声滤除。 排序晶圆图将基于对应的参考 DIE 的位置变换以及晶圆整体的分布情况，与模板晶圆图进行对齐。

参数
[out]sortedWaferMap输出排序后的晶圆图
返回值
error返回错误码
