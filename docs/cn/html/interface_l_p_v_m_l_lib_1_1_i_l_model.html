详细描述

该接口为机器学习模型，用于分类、检测、识别等任务。

首先需通过 Create() 创建指定类型的模型，然后通过 ILParameterized 接口调整参数。
下表列出当前支持的所有模型及其参数：

模型参数和使用说明
KNNK-Nearest Neighbors
参数名称使用说明类型取值范围默认值
"number of neighbors"用来预测的最近邻数量Integer1 ~ 645
"matching algorithm"用于最近邻搜索的算法Integersee LPVKNNMatchAlgoBrute-Force
"similarity metric"样本相似性度量方法Integersee LPVKNNDistanceEuclidean
"weight"查找到多个最近邻之后，计算最终投票结果时，确定每个最近邻权重的方法Integersee LPVKNNWeightBy distance
SVM支持向量机
参数名称使用说明类型取值范围默认值
"svm type"支持向量机模型的类型Integersee LPVSVMTypeC-Support
"cost"正则化参数 C，正则化的强度和 C 成反比Integer> 0 0.01
"complexity bound"该参数影响支持向量的最小数量，以及错分样本的最大数量，该参数越大意味着更多的支持向量（即模型更加复杂），以及对错分样本更多的容忍度Double0 ~ 10.5
"kernel"核函数Integersee LPVSVMKernelLinear
"g"核函数中的参数 gDouble> 0, 0 for auto-estimated0.1
"c"核函数中的参数 cDouble≥ 0 0.1
"n"核函数中的参数 nDouble> 0 3

示例代码

C++
// create a HOG feature
ILModelPtr svm(__uuidof(LModel));
svm->Create(LPVModelType::LPVModelSVM);
// change its parameters
svm->ParamI["svm type"] = LPVSVMType::LPVSVMTypeCSupport;
svm->ParamF["cost"] = 512;
svm->ParamI["kernel"] = LPVSVMKernel::LPVSVMKernelPolynomial;
svm->ParamF["c"] = 0.4;
llc->AddModel(svm); // add it to the classifier
C#
// create a HOG feature

svm.ParamF["c"] = 0.4;
llc.AddModel(svm); // add it to the classifier
