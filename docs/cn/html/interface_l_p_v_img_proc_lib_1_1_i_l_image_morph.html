详细描述

该接口类提供了多种形态学工具。

形态学工具通常使用一个预设的结构元，如矩形、圆形或十字等。对输入图像中的每个像素，形态学工具将这个像素的灰度值与其周围像素的灰度值进行比较， 将比较结果输出到输出图像的相应位置。结构元定义了像素周围需要比较的的邻域范围。 膨胀和腐蚀是最基础的形态学工具，其他工具由这两个工具组合得到。膨胀将比较的最大值作为结果，而腐蚀取比较的最小值。

使用这个接口，需要创建一个 LImageMorph 对象。

示例代码

C++
ILImageMorphPtr m_morph_op(__uuidof(LImageMorph));
m_morph_op->SetMorphShape(LPVMorphRect, 3, 3);
m_morph_op->Close(m_img, m_result_img);
C#
LImageMorph m_morph_op = new LImageMorph();
m_morph_op.SetMorphShape(LPVMorphShape.LPVMorphRect, 3, 3);
m_morph_op.Close(m_img, m_result_img);
