详细描述

该接口对于整个灵闪作业的配置，包括其中多个算法的配置。

通常通过 ILIBService::GetTaskConfig() 得到该 LIBTaskConfig 对象。 可以通过该即可访问作业中的单个算法工具的配置，并修改其具体参数。

示例代码

C++
ILIBAlgoConfigPtr algoInputs = m_ib_service->GetAlgoConfig(m_ib_task_id, L"algorithm name");
C#
LIBAlgoConfig algoInputs = m_ib_service.GetAlgoConfig(m_ib_task_id, "algorithm name");
