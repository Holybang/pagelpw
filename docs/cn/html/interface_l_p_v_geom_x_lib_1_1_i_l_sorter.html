详细描述

该接口类用于排序，可用于实现了 ILSortable 接口的对象的数组。

使用这个接口，需要创建一个 LSorter 对象。

示例代码

C++

Code snippets: snippets/Sort.cpp

CComSafeArray<ILRegionPtr> regions;
// add regions to the collection
ILSorterPtr sorter(__uuidof(LSorter));
sorter->SetSource2(regions);
// sort into a list of indexes
CComSafeArray<int> sortedIndexs = ((ILSortablesPtr)sorter)->SortTo(LPVSortBy::LPVSortByColumn,
LPVSortOrder::LPVSortOrderAscending, LPVPositionMode::LPVPosModeCenter);
// sort into an array
ILSortedArrayPtr sortedArray;
LPVErrorCode err = sorter->SortToArray(LPVPositionMode::LPVPosModeCenter, 0, 0, 1, &sortedArray);

阵列排序。后续可使用 ILSortedArray 接口进行行列索引、属性计算等。

参数
[in]posMode设置位置计算的方式。
[in]rowSpaceAppr输入的估计行间距，用于初始化排序矩阵。设置为 0，表示将基于输入对象的位置信息自动计算一个合适的数值。
[in]columnSpaceAppr输入的估计列间距，用于初始化排序矩阵。设置为 0，表示将基于输入对象的位置信息自动计算一个合适的数值。
[in]toleranceFactor该数值用于控制排序时的过滤行为，用于过滤输入中不完全落在阵列网格上的对象。 取值范围为 0 ~ 1，数值越小表示越严格的过滤。设置为 1，表示无过滤。
[out]sortedResult返回排序结果
返回值
error返回错误码
