详细描述

该接口类提供了用于图像类型转换的功能，可进行颜色空间转换和通道操作。

使用这个接口，需要创建一个 LImageConvert 对象。

示例代码

C++

Code snippets: snippets/ColorSpace.cpp

ILImageConvertPtr m_img_convert(__uuidof(LImageConvert));
m_img_convert->BGRToGray(m_color_img, m_gray_img);
m_img_convert->Convert(LPVColorSpaceBGR, LPVColorSpaceHSV, m_color_img, m_hsv_img);
C#

Code snippets: snippets/ColorSpace.cs

LImageConvert m_img_convert = new LImageConvert();
m_img_convert.BGRToGray(m_color_img, m_gray_img);
m_img_convert.Convert(LPVColorSpace.LPVColorSpaceBGR, LPVColorSpace.LPVColorSpaceHSV, m_color_img, m_hsv_img);
LPVCoreLib::LPVColorSpace
LPVColorSpace
该枚举表示图像的颜色空间，同时指定分量通道顺序。所有通道均归一化到 0 ~ 255 范围。

根据指定的各通道权重，混合 BGR 彩色图像的三个通道

\[ Gray = w0⋅B + w1⋅G + w2⋅R \]

参数

[in]w1第二个（绿色）通道的权重，取值范围在 -255 ~ 255
[in]w2第三个（红色）通道的权重，取值范围在 -255 ~ 255
[out]grayImg 返回灰度图像，尺寸与输入的彩色图像一致。
