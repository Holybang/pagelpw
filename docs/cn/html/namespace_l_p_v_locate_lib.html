详细描述

LPV 定位模块，提供经典的目标或边缘的定位算法。

该模块提供一些算法类，用于经典的目标或边缘定位任务。使用该模块中的类和算法，需按照以下方式引用该模块：

示例代码

C++
#include "LPVLocate.h"
C#
using LPVLocateLib;

◆ LPVLineDrawFlags

enum LPVLineDrawFlags

该枚举用于控制直线定位结果绘制的行为

枚举值
LPVLineDrawLine 

绘制直线

LPVLineDrawMidPoint 

绘制直线的中点

LPVLineDrawEndPoints 

绘制的端点

LPVLineDrawIndex 

绘制直线的索引，在其中点的右下方

LPVLineDrawDefault 

默认值，只绘制直线

LPVLineDrawAll 

绘制所有部分
