详细描述

该接口提供过滤数据斑点的功能

数据斑点过滤器用于按照属性特征过滤出合适的数据斑点对象。可按如下两种方法使用数据斑点过滤器:

  1. 配置过滤器再将其绑定到数据斑点分析算法上，通过 ILDataBlobAnalysis::AddBlobFilter().
  2. 自行调用过滤器的 ILDataBlobFilter::FilterResults() 接口，过滤数据斑点分析结果生成一个新的数据斑点对象集合。

支持在数据斑点分析算法中配置多个数据斑点过滤器，对于每一个候选数据斑点，若满足任意一个过滤器，则输出为最终的数据斑点分析结果。
支持在单个数据斑点过滤器中定义多个属性过滤，对于每一个候选数据斑点，若满足所有属性过滤设置，则是为满足该过滤器。

使用这个接口，需要创建一个 LDataBlobFilter 对象。

示例代码

C++
ILDataBlobFilterPtr m_data_blob_filter(__uuidof(LDataBlobFilter));
m_data_blob_filter->SetFilterFeature(LPVDataBlobFeatures::LPVDataBlobData, LPVAggregation::LPVAggMax, 100, 9999);
m_data_blob_result2 = m_data_blob_filter->FilterResults(m_data_blob_result);
C#
LDataBlobFilter m_data_blob_filter = new LDataBlobFilter();
m_data_blob_filter.SetFilterFeature(LPVDataBlobFeatures.LPVDataBlobData, LPVAggregation.LPVAggMax, 100, 9999);
m_data_blob_result2 = m_data_blob_filter.FilterResults(m_data_blob_result);
