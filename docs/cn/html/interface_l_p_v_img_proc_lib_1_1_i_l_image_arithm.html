详细描述

该接口类提供了图像类的数学运算功能。

使用这个接口，需要创建一个 LImageArithm 对象。

示例代码

C++
ILImageArithmPtr m_arithm_op(__uuidof(LImageArithm));
// m_result_img = m_img * 1.1 + 20
m_arithm_op->GainOffset(m_img, m_result_img, 1.1, 20);
// m_result_img = avg(m_img_list)
m_arithm_op->Blend(m_img_list, LPVAggregation::LPVAggAvg, m_result_img);
C#
LImageArithm m_arithm_op = new LImageArithm();
// m_result_img = m_img * 1.1 + 20
m_arithm_op.GainOffset(m_img, m_result_img, 1.1, 20);
// m_result_img = avg(m_img_list)
m_arithm_op.Blend(m_img_list, LPVAggregation::LPVAggAvg, m_result_img);

◆ Blend()

void Blend ( ILImageList *  imgList,
LPVAggregation  aggType,
ILImage *  result 
)

多帧图像融合用于将多张同一场景的图像融合为一张。在不同的场景，通过使用不同的聚合方式达到不同目的。

聚合方式 效果
均值 降低图像噪声，在光照条件不佳、亮度不断变化的场景下，获得更稳定的图像质量。
最小值 减少亮部区域，例如，用于减少过曝区域。
最大值 减少暗部区域，减少阴影
标准差 结果图像中的较亮像素表示在多帧图像中该位置的像素值变化较大

注意：图像融合不支持其他为指明的聚合方式。

参数
[in]imgList输入用于融合的图像队列
[in]aggType融合方式
[out]result输出融合结果图像

Example: snippets/ImageBlend.cpp, snippets/ImageBlend.cs
