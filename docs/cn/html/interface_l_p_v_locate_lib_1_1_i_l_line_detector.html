详细描述

该接口提供直线定位的功能。

直线定位用于在输入图像中和指定的区域中，定位一条或多条直线边缘。
该算法具有高实时性、高亚像素精度等特点，其局限性在于其定位直线的角度在给定区域的扫描方向的 (-10, +10) 范围内。 因此，使用该算法需要指定目标直线的大致方向。

使用这个接口，需要创建一个 LLineDetector 对象。

示例代码

C++
ILLineDetectorPtr line_detector(__uuidof(LLineDetector));
line_detector->Polarity = LPVPolarity::LPVPolarityEitherEdge;
LPVErrorCode err = line_detector->Detect(img, roi, &line_results);
if (err != LPVErrorCode::LPVNoError) return;
