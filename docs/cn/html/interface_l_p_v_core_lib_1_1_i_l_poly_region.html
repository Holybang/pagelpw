详细描述

该接口对应一个多边形区域对象。

使用这个接口，需要创建一个 LPolyRegion 对象。

示例代码

C++
ILPolyRegionPtr m_poly_roi(__uuidof(LPolyRegion));
m_poly_roi->SetPlacement(20, 20, 100);
C#
LPolyRegion m_poly_roi = new LPolyRegion();
m_poly_roi.SetPlacement(20, 20, 100);
