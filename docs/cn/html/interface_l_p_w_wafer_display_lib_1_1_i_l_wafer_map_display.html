详细描述

该接口提供显示控件功能。

通常可通过 Visual Studio 的工具箱，将显示控件加入到应用程序窗口中。 该接口提供管理显示窗口内场景的功能，用于设置场景中的图像，增加删除场景中的 region、几何形状和算法结果等。

控件事件请查看 ILWaferMapDisplayEvents 。

示例代码

C++
// set image to the control
m_displayCtrl->SetImage(img);
// add a rectangle region to the control
ILRectRegionPtr roi(__uuidof(LRectRegion));
roi->SetPlacement(10, 10, 200, 100);
m_displayCtrl->AddRegion(roi, true); // it's interactive
// add a circle to the control
ILCirclePtr circle(__uuidof(LCircle));
circle->Set(100, 50, 50);
m_displayCtrl->AddObject(circle, 0); // use default draw flags
// add the pattern matching result to the control
m_displayCtrl->AddObject(match_results, (LPVPatDrawFlags)(LPVPatDrawFlags::LPVPatDrawBoundingRect | LPVPatDrawFlags::LPVPatDrawCenter));
