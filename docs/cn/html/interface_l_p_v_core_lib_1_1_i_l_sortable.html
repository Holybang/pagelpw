详细描述

该接口类用于表示一个可排序对象。

所有实现了该接口的 LPV 类，都用可以使用 LSorter 进行排序。 排序结果记录在 Index 中。

以下列出了所有实现该接口的可排序 LPV 类。

LPVCoreLibLRegion, LRectRegion, LRotRectRegion, LCircleRegion, LPolyRegion, LMaskRegion, LAnnulusRegion, LEllipseRegion, LAnnulusSectorRegion
LPVGeomLibLPoint, LPoints, LCircle, LLine, LPolygon, LRect, LEllipse, LArc
LPVPatLibLMatchResult, LMatchResults

示例代码

C++
CComSafeArray<ILRegionPtr> regions;
// add regions to the collection
ILSorter sorter(__uuidof(LSorter));
sorter->SetArray(regions);
CComSafeArray<int> sortedIndexs = ((ILSortablesPtr)sorter)->SortTo(LPVSortBy::LPVSortArrayByColumn,
LPVSortOrder::LPVSortOrderAscending, LPVPositionMode::LPVPosModeCenter);
for (int i = 0; i < sortedIndexs.GetCount(); ++i) {
int index = sortedIndexs.GetAt(i);
ILRegionPtr sortedRegion = regions.GetAt(index);
}
C#
LRegion[] regions;
// add regions to the collection
LSorter sorter = new LSorter();
sorter.SetArray(regions);
int[] sortedIndexs = (int[])((LSortables)sorter).SortTo(regions, LPVSortBy.LPVSortArrayByColumn,
LPVSortOrder.LPVSortOrderAscending, LPVPositionMode.LPVPosModeCenter);
foreach (int index in sortedIndexs) {
LRegion sortedRegion = regions[index];
}
