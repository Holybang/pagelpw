详细描述

该接口类提供了图像边缘增强的功能。

使用这个接口，需要创建一个 LEdgeFilter 对象。

示例代码

C++
ILEdgeFilterPtr m_edge_filter_op(__uuidof(LEdgeFilter));
m_edge_filter_op->SetEdgeKernel(LPVSobel, 5, 5);
m_edge_filter_op->GradientFull(m_img, m_result_img);
C#
LEdgeFilter m_edge_filter_op = new LEdgeFilter();
m_edge_filter_op.SetEdgeKernel(LPVEdgeKernel.LPVSobel, 5, 5);
m_edge_filter_op.GradientFull(m_img, m_result_img);

设置图像边缘滤波器所使用的滤波核类型核尺寸。

参数
[in]kType核类型。
[in]kWidth滤波核的宽度，需要是一个奇数，如 1, 3, 5, 7 等。 若核尺寸设置为 1，将退化为 \( (-1, 1) \)，指定的核类型不起作用。
[in]kHeight滤波核的高度，需要是一个奇数，如 1, 3, 5, 7 等。若设置为 0，则使用与宽度相同。
