详细描述

该接口类提供了多种多边形滤波、剪裁、修改等工具，可用于对 LPolygon 对象进行平滑、简化、布尔运算、膨胀、收缩等操作。

使用这个接口，需要创建一个 LPolygonClipper 对象。

示例代码

C++

Code snippets: snippets/PolygonClipper.cpp

ILPolygonClipperPtr clipper(__uuidof(LPolygonClipper));
ILPolygonPtr smoothPoly = clipper->Smooth(blobContour, 30);
ILPolygonPtr resultPoly = clipper->Simplify(smoothPoly, 0, 20, 150);
C#

Code snippets: snippets/PolygonClipper.cs

LPolygonClipper clipper = new LPolygonClipper();
LPolygon smoothPoly = clipper.Smooth(blobContour, 30);
LPolygon resultPoly = clipper.Simplify(smoothPoly, 0, 20, 150);

平滑多边形，迭代地对多边形边的法向进行均值滤波并以此优化更新顶点位置 该方法相对于普通的均值滤波（即 Smooth() ），更容易获得形状更好的多边形，且不易造成过度平滑，但耗时更长。

参数
[in]poly输入的多边形对象
[in]filterNbRng滤波邻域 \( (-filterNbRng, filterNbRng) \)，即沿着多边形的边长距离小于 filterNbRng 的顶点（包含临界处的插值）将参与法向滤波。 滤波邻域越大，滤波结果越平滑。
[in]updateNbRng更新邻域 \( (-updateNbRng, updateNbRng) \)，即沿着多边形的边长距离小于 filterNbRng 的顶点（包含临界处的插值）将参与更新多边形顶点。 更新邻域通常小于滤波邻域，合适的更新邻域可加速迭代过程的收敛，使得迭代提前结束。推荐使用 2 ~ 5，默认设置为 0 即使用滤波邻域的 10% 作为更新邻域。
[in]maxIterCount最大迭代次数。默认值为 30.
[in]stopCriteria迭代终止条件。默认值为 0.001 。若当前迭代所有顶点的最大移动距离小于设定值，或者前后两次迭代移动距离的差值小于 10，则迭代终止，即: \( m_i < criteria || (m_i - m_{i-1}) / m_{i-1} < 0.1% \)
返回值
resultPoly返回多边形结果对象

double  angleThreshold

简化多边形，基于给定的面积或角度阈值，清除多边形上多余的顶点。

参数
[in]poly输入的多边形对象
[in]maxVertexes结果的最大顶点数量。可指定为 0，即剔除所有满足阈值的顶点。
[in]areaThreshold面积阈值。面积指的是删除某个顶点（及其相邻的所有已删除顶点）所造成的总缺失面积。 若该总面积小于指定的面积阈值，则该顶点为可删除顶点。
[in]angleThreshold角度阈值。角度指的是某个顶点以及与之相邻的两个其他顶点所组成的夹角。 若该角度小于角度阈值，则该顶点为可删除顶点。角度阈值的取值范围在 0 ~ 180 之间。 若同时指定了面积阈值和角度阈值，则面积和角度加权后同时参与简化过程。
若面积阈值和角度阈值均为 0，则简化过程退化为清除所有与相邻顶点共线的冗余顶点的操作。
返回值
resultPoly返回多边形结果对象
参见
SimplifyDP()
