详细描述

该接口表示单个刻线检测结果，可获取其具体属性。

通常通过 ILKerfDetector::Detect1() 或者 ILKerfDetector::Detect2() 函数获得 LKerfResult 对象。 可以使用该接口访问单个刻线检测对象的具体信息，如边线位置、刻线能量评分等，还可以调用绘制函数将该检测结果绘制在界面上。

示例代码

C++
ILKerfResultPtr result;
LPVErrorCode err = m_kerftool->Detect1(m_img, m_rotroi, &result);
if (err != LPVNoError) return; // no result or failed
double kerfWidth = result->LineUp->Distance(result->LineDown);

绘制检测分数曲线到给定的设备。该曲线表示在沿着检测区域的方向（x 轴）上检测分数（y 轴）的分布情况。 通常用于为刻线定位调试合适的对比度和涂胶区域阈值。

参数
[in]hdc绘制设备，可为窗口、位图等的上下文句柄。
[in]drawFlags用于控制绘制的内容，查看 LPVChartDrawFlags。
[in]xGridStep绘制 x 轴的步长，默认设置为 0，表示按照分数曲线数据自动选择合适的步长。
[in]yGridStep绘制 y 轴的步长，默认设置为 0，表示按照分数曲线数据自动选择合适的步长。
