详细描述

该接口提供一组斑点分析结果的集合，包含一个或多个斑点。

通常通过调用 ILBlobAnalysis::Build() 得到该检测结果集合。 可以通过该接口访问其中的单个检测结果，绘制检测结果到给定的设备。

示例代码

C++
ILBlobResultsPtr blob_results;
LPVErrorCode err = blob_ana->Build(img, roi, &blob_results);
if (err != LPVErrorCode::LPVNoError) return;
int c = blob_results->Count();
for (int i = 0; i < c; ++i) {
ILBlobPtr blob = blob_results->Item(i);
// handle each blob result
}
