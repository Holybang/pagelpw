详细描述

该接口提供单个模板匹配结果，可获取其具体属性。

通常通过 ILMatchResults::Item() 函数获得 LMatchResult 对象。可以使用该接口访问单个模板匹配结果的具体信息，如中心、分数等， 还可以调用绘制函数将单个对象绘制在界面上。

示例代码

C++
ILMatchResultPtr result = m_match_results->Item(i);
double x = result->Center->X;
C#
LMatchResult result = m_match_results.Item(i);
double x = result.Center.X;
