详细描述

该接口对应一个可绘制的对象。

使用这个接口，需要创建一个 LText 对象。

示例代码

C++
ILTextPtr m_text(__uuidof(LText));
m_text->Text = L"My Text";
// put the text on (200, 200), center-aligned
m_text->PosX = 200;
m_text->PosY = 200;
m_text->Alignment = LPVAlignCenter;
// change the color to red
m_sys->SetFontColor(m_text, m_sys->GetColorRef(255, 0, 0));
// draw the text in the display control
m_display->AddObject(m_text, 0);
// or, draw it in your own device context
m_text->Draw(hdc, zoomX, zoomY, panX, panY);
