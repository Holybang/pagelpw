详细描述

该接口类提供了像素级的图像比较功能。

使用这个接口，需要创建一个 LImageCompare 对象。

2.1.0

C++

Code snippets: snippets/CrossMarkInspect.cpp

ILImageComparePtr imgCmp = LImageCompare::Create();
imgCmp->PixelTolerance = 20;
LPVErrorCode err = imgCmp->SetBaseline(img, roi);
if (err != LPVErrorCode::LPVNoError) return; // handle error
ILImagePtr binResult = LImage::Create();
err = imgCmp->Compare(img, roi, nullptr, binResult); // get the binary comparison result only
if (err != LPVErrorCode::LPVNoError) return; // handle error
