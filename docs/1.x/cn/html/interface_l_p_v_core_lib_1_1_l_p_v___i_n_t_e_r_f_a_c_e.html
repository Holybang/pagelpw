详细描述

该接口类开放了可以改变一个可绘制对象的图形设置的接口。

该接口类用于对一个可排序对象集合进行排序或重整为给定的顺序。

该接口类用于表示一个可排序对象。

所有提供了绘制功能的 LPV 类都实现这个类定义的所有接口。 使用这个接口类，需将 LPV 的类转换成 ILDrawable 接口类，然后通过该接口修改其描边笔、笔刷、字体等绘制属性。

以下列出了所有实现该接口的可绘制 LPV 类。
