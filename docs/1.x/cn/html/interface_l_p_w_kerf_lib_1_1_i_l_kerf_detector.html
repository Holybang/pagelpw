详细描述

该接口提供刻线检测和定位的功能。

刻线定位用于在输入图像中和指定的区域中，定位一组（Wide）或两组（Narrow）条刻线边缘。
该算法具有高精度、高抗噪性和稳定性，对于刻线上的涂胶区域不敏感，可应对不同的背景和检测环境。

使用这个接口，需要创建一个 LKerfDetector 对象。

示例代码

C++
ILKerfDetectorPtr m_kerftool(__uuidof(LKerfDetector));
m_kerftool->KerfWidth = 10;
m_kerftool->KerfDistance = 40;
m_kerftool->ContrastThreshold = 2;
ILKerfResultPtr result1, result2;
LPVErrorCode err = m_kerftool->Detect2(m_img, m_rotroi, &result1, &result2);
if (err != LPVNoError) return; // no result or failed
