详细描述

该接口类为显示控件的基类，提供基本控件对应的接口。

这是所有显示控件对象的基类，对于不同的显示内容，需选择不同的显示控件。 使用这个接口，需要创建一个显示控件对象，如:

◆ SaveSceneToFile()

int SaveSceneToFile ( BSTR  filename,
double  zoomX,
double  zoomY 
)

保存整个场景到文件，包括图像、所有区域和绘制对象，以彩色图像保存。 图像尺寸与缩放后的场景尺寸相同。图像格式按照所提供的扩展名自动选择 (.bmp, .png, .jpg, etc.)。

参数
[in]filename文件路径，可为绝对或相对路径，包括所在文件夹，文件名和扩展名。
[in]zoomXx 方向的缩放系数，默认设置为 1，即真实尺寸 100%
[in]zoomYy 方向的缩放系数，默认设置为 0，即与 x 方向的缩放系数相同，也就是等比缩放。
返回值
error返回错误码，查看 LPVErrorCode
