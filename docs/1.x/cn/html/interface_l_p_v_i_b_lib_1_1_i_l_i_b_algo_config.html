详细描述

该接口对应灵闪算法工具的配置，包括其名称、输入参数等。

通常通过 ILIBTaskConfig 接口，或者通过 ILIBService::GetAlgoConfig()，获取 LIBAlgoConfig 对象。 可以通过该接口访问算法单个输入参数配置，并修改其具体值。

示例代码

C++
ILIBAlgoConfigPtr algoInputs = m_ib_service->GetAlgoConfig(m_ib_task_id, L"algorithm name");
int dataCount = algoInputs->Count();
for (int i = 0; i < dataCount; ++i) {
ILIBDataPtr ibData = outputResult->Item(i);
double oldValue = ibData->ToDouble(); // read old value
ibData->FromDouble(newValue); // assign new value
algoInputs->Config[ibData->Name] = ibData; // set the changed data
}
// set the changed configuration to IntelliBlink service
m_ib_service->SetAlgoConfig(m_ib_task_id, L"algorithm name", algoInputs);
