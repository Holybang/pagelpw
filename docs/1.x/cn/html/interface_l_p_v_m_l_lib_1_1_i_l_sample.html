详细描述

该接口类表示单个样本，通常由一个图像和可选的区域对象组成。

样本通常由样本数据库 LSampleDatabase 组织和管理。 使用这个接口，需要创建一个 LSample 对象，或通过样本数据库获取该接口。

示例代码

C++
// create the sample then add to the database
ILSamplePtr sample1(__uuidof(LSample));
sample1->Create(img, region);
sampleDb->AddSample(LPVSampleType::LPVSampleTrain, sample1);
// create then get back from the database
std::wstring sampleName = (wchar_t*)sampleDb->AddSampleImage(LPVSampleType::LPVSampleTrain, img, region);
ILSamplePtr sample2 = db2->GetSample(sampleName.c_str());
C#
// create the sample then add to the database
LSample sample1 = new LSample();
sample1.Create(img, region);
sampleDb.AddSample(LPVSampleType.LPVSampleTrain, sample1);
// create then get back from the database
String sampleName = sampleDb.AddSampleImage(LPVSampleType.LPVSampleTrain, img, region);
LSample sample2 = db2.GetSample(sampleName);

◆ Create()

void Create ( ILImage *  img,
ILRegion *  sampleRegion 
)

基于输入的图像和区域，创建样本。若输入区域不为空，样本图像将基于区域尺寸裁剪获得该区域的子图像。 若输入区域为空，整个输入图像作为一个样本。

参数
[in]img输入的图像
[in]sampleRegion输入的区域对象，可为空
