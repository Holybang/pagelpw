详细描述

该接口类表示单行文本对象，用于获取其具体属性。

通常通过 ILOCRResult::ItemLine() 函数获得 LOCRLine 对象。可以使用该接口访问单行文本对象的具体信息，如位置、识别结果、局部图像等， 还可以调用绘制函数将单行文本对象绘制在界面上。

示例代码

C++
ILOCRLinePtr line_result = ocr_result->Item(i);
std::wstring line_value = line_result->String;
C#
LOCRLine line_result = ocr_result.Item(i);
String line_value = line_result.String;
