详细描述

该接口类提供了多种图像滤波工具。

图像滤波工具是计算机视觉系统中十分常用的一类图像预处理方法，用于在目标识别、缺陷检测等任务之前或过程中，对图像进行平滑、去噪、增强或归一化等操作。 大多数的图像滤波算法使用线性滤波器，即输出的像素值是输入像素的加权求和：

\[ I'(x,y) = \sum_{i,j} f(x+i, y+j) k(i,j) \]

\( k(i,j) \) 为滤波器所使用的核。

使用这个接口，需要创建一个 LImageFilter 对象。

示例代码

C++
ILImageFilterPtr m_img_filter_op(__uuidof(LImageFilter));
m_img_filter_op->SetKernelSize(5, 5);
m_img_filter_op->Gaussian(m_img, m_result_img);
C#
LImageFilter m_img_filter_op = new LImageFilter();
m_img_filter_op.SetKernelSize(5, 5);
m_img_filter_op.Gaussian(m_img, m_result_img);

◆ Gaussian()

void Gaussian ( ILImage *  img,
ILImage *  result 
)

使用高斯滤波器模糊图像。高斯滤波工具将计算 x、y 方向的 \( \texttt{ksize} \times 1 \) 高斯核：

\[ G_i= \alpha *e^{-(i-( \texttt{ksize} -1)/2)^2/(2* \texttt{sigma}^2)}, \]

其中 \( i=0..\texttt{ksize}-1 \) ， \( \alpha \) 是自动设置的缩放系数使得 \( \sum_i G_i=1 \) ， \( sigma \) 基于核尺寸得到： \( ksize \times 0.6 \) .

参数
[in]img输入图像
[out]result输出结果图像

◆ SetKernelSigma()

void SetKernelSigma ( double  kSigma,
double  kGain 
)

设置高斯类滤波核的 Sigma 和 Gain 值

参数
[in]kSigma用于控制高斯滤波核的平滑程度。
若设置为 0（默认），则由核尺寸自动计算为： \( 1/6 \)
若核尺寸设置为 0，则由 Sigma 值自动计算合适的核尺寸，约为 \( 6 \times kSigma \)
[in]kGain用于控制核的尺寸。若核尺寸已由 SetKernelSize() 方法指定，则忽略 kGain 值。
对于 Sharpen()，kGain 还用于控制锐化效果的强度。
参见
SetKernelSize(), HighPass(), Gaussian(), Sharpen()

◆ SetKernelSize()

void SetKernelSize ( int  kWidth,
int  kHeight 
)

设置滤波工具中使用的核尺寸，默认设置为 \( 3 \times 3 \) 大小。 尺寸更大的滤波核尺寸通常会带来更强的滤波效果，但速度更慢。
若核尺寸设置为 0，则由 Sigma 值自动计算合适的核尺寸。

参数
[in]kWidth滤波核的宽度，需要是一个奇数，如 3, 5, 7 等。
[in]kHeight滤波核的高度，需要是一个奇数，如 3, 5, 7 等。若设置为 0，则使用与宽度相同。
参见
SetKernelSigma()
