详细描述

该接口类为可序列化 LPV 对象的基类。

该接口类提供了修改 LPV 系统全局参数的功能和一些杂项帮助函数。

使用这个接口，需要创建一个 LSystem 对象。

示例代码

C++
ILSystemPtr sys = LSystem::Create();
sys->SetDrawPointSize(20);
C#
LSystem sys = new LSystem();
sys.SetDrawPointSize(20);

@add_toggle_com

ILSystemPtr sys = LSystem::Create();
// or, ILSystemPtr sys(__uuidof(LSystem)), for absence of built-in Create functions (before 2.2.0)
sys->SetDrawPointSize(20);

◆ CheckLib()

BOOL CheckLib ( LPV_STRING  libName)

检查某个模块是否合法授权。授权后，可以正常使用该模块中的类和算法函数。
通常按照如下方式调用：

C++
bool checkResult = sys->CheckLib(L"lpvCore");
if (!checkResult) return; // return if check license failed for lpvCore library
C#
bool checkResult = sys.CheckLib("lpvCore");
if (!checkResult) return; // return if check license failed for lpvCore library

@add_toggle_com

VARIANT_BOOL checkResult = sys->CheckLib(L"lpvCore");
if (!checkResult) return; // return if check license failed for lpvCore library
参数
[in]libName输入模块的名称。 该名称不区分大小写，且可省略 "lpv" 前缀，如 "lpvCore.dll", "lpvCore", "core", "CORE".
返回值
val若检查成功，返回 True，否则返回 False

◆ CheckVersion()

BOOL CheckVersion ( long  majorVersion,
long  minorVersion,
long  microVersion 
)

检查编译时和运行时所使用的 LPW 版本兼容性。通常按照如下方式调用：

C++
bool checkResult1 = sys->CheckVersion(1, 13, 1); // the required version is 1.13.1
if (!checkResult1) return; // return if check version failed
bool checkResult2 = sys->CheckVersion(LPVMajorVersion, LPVMinorVersion, LPVMicroVersion); // use the built-in constants
if (!checkResult2) return; // return if check version failed
C#
bool checkResult1 = sys.CheckVersion(1, 13, 1); // the required version is 1.13.1
if (!checkResult1) return; // return if check version failed
bool checkResult2 = sys.CheckVersion(LPVMajorVersion, LPVMinorVersion, LPVMicroVersion); // use the built-in constants
if (!checkResult2) return; // return if check version failed

@add_toggle_com

VARIANT_BOOL checkResult1 = sys->CheckVersion(1, 13, 1); // the required version is 1.13.1
if (!checkResult1) return; // return if check version failed
VARIANT_BOOL checkResult2 = sys->CheckVersion(LPVMajorVersion, LPVMinorVersion, LPVMicroVersion); // use the built-in constants
if (!checkResult2) return; // return if check version failed

若检查失败，用户需要重新安装正确的运行版本，或重新基于更新的版本编译。推荐在编译和运行时使用相同版本。

参数
[in]majorVersion编译时的版本号第一位
[in]minorVersion编译时的版本号第二位
[in]microVersion编译时的版本号第三位
返回值
val若检查成功，返回 True，否则返回 False
