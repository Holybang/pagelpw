详细描述

该接口提供过滤斑点的功能

斑点过滤器用于按照属性特征过滤出合适的斑点对象。可按如下两种方法使用斑点过滤器:

  1. 配置过滤器再将其绑定到斑点分析算法上，通过 ILBlobAnalysis::AddBlobFilter().
  2. 自行调用过滤器的 ILBlobFilter::FilterResults() 接口，过滤斑点分析结果生成一个新的斑点对象集合。

支持在斑点分析算法中配置多个斑点过滤器，对于每一个候选斑点，若满足任意一个过滤器，则输出为最终的斑点分析结果。
支持在单个斑点过滤器中定义多个属性过滤，对于每一个候选斑点，若满足所有属性过滤设置，则是为满足该过滤器。

使用这个接口，需要创建一个 LBlobFilter 对象。

示例代码

C++
ILBlobFilterPtr blob_filter(__uuidof(LBlobFilter));
blob_filter->SetFilterFeature(LPVBlobFeatures::LPVBlobArea, 1000, -1);
blob_result2 = blob_filter->FilterResults(blob_result);
C#
LBlobFilter blob_filter = new LBlobFilter();
blob_filter.SetFilterFeature(LPVBlobFeatures.LPVBlobArea, 1000, -1);
blob_result2 = blob_filter.FilterResults(blob_result);
