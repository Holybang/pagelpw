LColor BGColor [get, set]
 背景色，用于填充计算过程中未覆盖的像素，如 Rotate(), TileX()。默认设置为黑色。
 
double ResultGain [get, set]
 对运算结果进行增益，放大或缩小数值分布范围。需要是一个非零值，默认值为 1，即不缩放。
 
double ResultOffset [get, set]
 将运算结果加上偏移值，在增益 ResultGain 之后进行。默认值为 0，即无偏移。
