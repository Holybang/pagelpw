Detailed Description

This interface expose APIs to create/modify a mask region object.

White pixels on the mask image represent pixels should be included in the region, while black pixels represent the ones should be excluded. To use this interface, you should create a LMaskRegion object.

Example Code

C++
ILMaskRegionPtr mask_roi(__uuidof(LMaskRegion));
mask_roi->SetPlacement(mask_img, 0, 0);
C#
LMaskRegion mask_roi = new LMaskRegion();
mask_roi.SetPlacement(mask_img, 0, 0);
