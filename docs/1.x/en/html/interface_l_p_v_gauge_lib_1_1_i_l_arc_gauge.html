Detailed Description

This interface provide functionalities of arc inspection via the gauge.

The arc gauge is used to measure position and size of the imaginary circle formed by the circular edges. Edge points are sampled along the given region using equally spaced point measurement gauges, and then fit to a circle. The start and end point of the result arc is then calculated based on the intersection of the imaginary circle and the region, to get the final result arc.

To use this interface, you should create a LArcGauge object.

Example Code

C++
ILArcGaugePtr arcGauge(__uuidof(LArcGauge));
arcGauge->Polarity = LPVPolarity::LPVBlack2White;
arcGauge->AcceptScore = 40;
ILShapeGaugeResultPtr arcGaugeResult;
LPVErrorCode err = arcGauge->Detect(img, annulus_sector_roi, &arcGaugeResult);
if (err != LPVErrorCode::LPVNoError) return;
