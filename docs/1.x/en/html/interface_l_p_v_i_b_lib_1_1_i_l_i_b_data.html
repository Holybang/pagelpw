Detailed Description

This interface holds an IntelliBlink data, it could be the input parameter or the output result.

To use this interface, you should create a ILIBData object.

Example Code

C++
// as result
ILIBDataPtr ibDataResult = algoResult->Result[L"Line"];
ILLinePtr line = ibDataResult.ToLine();
// as input
ILIBDataPtr ibDataInput(__uuidof(LIBData));
ibDataInput->FromDouble(12.34);
algoInputs->Config[L"Score"] = ibDataInput;
C#
// as result
LIBData ibDataResult = algoResult.Result["Line"];
LLine line = ibDataResult.ToLine();
// as input
LIBData ibDataInput = new LIBData();
ibDataInput.FromDouble(12.34);
algoInputs.Config["Score"] = ibDataInput;
