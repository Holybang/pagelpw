Detailed Description

This interface expose APIs for basic and advanced geometry calculation functionalities.

To use this interface, you should create a LGeomOp object.

Example Code

C++
ILGeomOpPtr geomOp(__uuidof(LGeomOp));
ILRectPtr resultRect = geomOp->InnerRect(mask);
ILRectPtr resultSquare = geomOp->InnerSquare(mask);
C#
LGeomOp geomOp = new LGeomOp();
LRect resultRect = geomOp.InnerRect(mask);
LRect resultSquare = geomOp.InnerSquare(mask);
