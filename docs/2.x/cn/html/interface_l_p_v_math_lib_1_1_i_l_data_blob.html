详细描述

该接口对应一个数据斑点，用于获取数据斑点的具体属性

通常通过调用 ILDataBlobResults::Item() 得到该 LDataBlob 对象。 可以通过该接口计算感兴趣的数据斑点属性特征，访问数据斑点信息。

示例代码

C++
ILDataBlobPtr dataBlob = mBlobResults->Item(i);
double avg = dataBlob->GetFeature(LPVDataBlobFeatures::LPVDataBlobData, LPVAggregation::LPVAggAvg);
C#
LDataBlob dataBlob = mBlobResults.Item(i);
double avg = dataBlob.GetFeature(LPVDataBlobFeatures.LPVDataBlobArea, LPVAggregation.LPVAggAvg);
