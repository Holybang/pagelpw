详细描述

该接口类提供了阈值相关的工具

阈值工具是一类经典的图像分割方法，将一个图像分为需要分析的前景区域和背景。 分割过程基于图像中像素灰度在前景和背景区域的分布，对每一个像素，将其与给定的阈值或自动适应的阈值进行比较，以此区分前景和背景像素。

使用这个接口，需要创建一个 LImageThreshold 对象。

示例代码

C++
ILImageThresholdPtr thresholdOp = LImageThreshold::Create();
thresholdOp->SetThreshold(100, 200);
thresholdOp->Binarize(img, resultImg);
C#
LImageThreshold thresholdOp = new LImageThreshold();
thresholdOp.SetThreshold(100, 200);
thresholdOp.Binarize(img, resultImg);
COM
ILImageThresholdPtr thresholdOp = LImageThreshold::Create();
// or, ILImageThresholdPtr thresholdOp(__uuidof(LImageThreshold)), for absence of built-in Create functions (before 2.2.0)
thresholdOp->SetThreshold(100, 200);
thresholdOp->Binarize(img, resultImg);

ILImage *  result 
)

二值化输入图像并输出结果图像。
若所设置的阈值范围的下限小于上限，输入图像中灰度值在阈值范围内的像素，在结果图中将被设置为前景（最大数值，8 位图为 255，16 位图为 65535），反之，则被设置为背景（0）。
若所设置的阈值范围的下限大于上限，输入图像中灰度值在阈值范围内的像素，在结果图中将被设置为背景（0），反之，则被设置为前景（最大数值）。

参数
[in]img 输入图像
[out]result 输出结果图像

裁剪输入图像并输出结果图像。

◆ SetThresholdAdaptGlobal()

void SetThresholdAdaptGlobal ( int  lbBias,
int  ubBias 
)

设置并使用全局自适应阈值。
自动基于整幅图像上的像素灰度值分布计算一个合适的阈值，然后按照用户设置的上下偏置，生成阈值范围。 全局自适应阈值范围的下限为 \( t - lbBias \) ，上限为 \( t + ubBias \) 。

参数
[in]lbBias基于全局自适应阈值的下限修正值，用于生成阈值范围的下限。
[in]ubBias基于全局自适应阈值的上限修正值，用于生成阈值范围的上限。

◆ SetThresholdAdaptLocal()

void SetThresholdAdaptLocal ( int  blockWidth,
int  blockHeight,
int  lbBias,
int  ubBias 
)

设置并使用局部自适应阈值。
自动基于每个像素所在的 \( blockWidth \times blockHeight \) 范围内的像素灰度值分布，为每个位置计算一个合适的阈值。 该位置的局部自适应阈值范围的下限为 \( t - lbBias \) ，上限为 \( t + ubBias \) 。

参数
[in]blockWidth局部区域的宽度，需要是一个奇数，如 3, 5, 7 等。
[in]blockHeight局部区域的高度，需要是一个奇数，如 3, 5, 7 等。
[in]lbBias基于该位置的局部自适应阈值的下限修正值，用于生成阈值范围的下限。
[in]ubBias基于该位置的局部自适应阈值的上限修正值，用于生成阈值范围的上限。

◆ Stretch()

void Stretch ( ILImage *  img,
ILImage *  result 
)

拉伸输入图像并输出结果图像。
若像素的灰度值在所设置的阈值范围内，则将其拉伸到 0 到 255 的数值；超出的像素设置为 0 或最大数值.
例如，对于一张 8 位图像，如若阈值范围设置为 [100, 200]，像素灰度在 [0, 100] 设置为 0，像素灰度在 [200, 255] 设置为 255 像素灰度在 [101, 199] 之间的按照以下公式计算：

\[ (p - 100) \times 255 / (200 - 100) \]

参数
[in]img 输入图像
[out]result 输出结果图像
