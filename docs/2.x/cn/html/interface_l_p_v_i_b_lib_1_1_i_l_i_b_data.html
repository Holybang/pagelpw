详细描述

该接口对应一个来自灵闪服务的数据，可能是输入参数数据或输出的结果数据。

使用这个接口，需要创建一个 ILIBData 对象。

示例代码

C++
// as result
ILIBDataPtr ibDataResult = algoResult->GetResult(L"Line");
ILLinePtr line = ibDataResult.ToLine();
// as input
ILIBDataPtr ibDataInput = LIBData::Create();
ibDataInput->FromDouble(12.34);
algoInputs->PutConfig(L"Score", ibDataInput);
C#
// as result
LIBData ibDataResult = algoResult.Result["Line"];
LLine line = ibDataResult.ToLine();
// as input
LIBData ibDataInput = new LIBData();
ibDataInput.FromDouble(12.34);
algoInputs.Config["Score"] = ibDataInput;
COM
// as result
ILIBDataPtr ibDataResult = algoResult->Result[L"Line"];
ILLinePtr line = ibDataResult.ToLine();
// as input
ILIBDataPtr ibDataInput = LIBData::Create();
// or, ILIBDataPtr ibDataInput(__uuidof(LIBData)), for absence of built-in Create functions (before 2.2.0)
ibDataInput->FromDouble(12.34);
algoInputs->Config[L"Score"] = ibDataInput;

注：该接口在 Linux / MacOS 平台暂不可用。
