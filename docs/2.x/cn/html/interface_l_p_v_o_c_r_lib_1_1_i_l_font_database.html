详细描述

该接口类表示用于字符识别的字库对象。

字库用于管理字符样本，并训练基于机器学习技术的分类模型以用于字符识别。 该算法类提供了多种基于图像或几何属性的专用于字符识别的特征，并支持高级自动特征选择和模型优化（如果有足够的字符样本）。 字库可构建到指定的文件目录，并被多个 OCR 工具共享。

使用这个接口，需要创建一个 LFontDatabase 对象。

示例代码

C++
ILFontDatabasePtr fontDb = LFontDatabase::Create();
fontDb->Init(L"ABC Font", L"D:/MyFontDB/");
// add samples
fontDb->AddOCRResult(segResult, L"ABC\nDEF\n123");
fontDb->AutoFeature = true; // auto-selected
LPVErrorCode err = fontDb->Train();
if (err != LPVErrorCode::LPVNoError) return;
// set the font database to OCR tool
ocrTool->FontDatabase = fontDb;
