BOOL AllowDup [get, set]
 设置在解码过程中是否进行剔除重复内容的条码结果。默认设置为 False， 即若检测区域中有多个编码内容相同的条码，只输出第一个检测结果（忽略其他）。
 
BOOL CheckSum [get, set]
 设置在解码过程中是否校验条码。默认设置为 True
 
int MaxCount [get, set]
 设置检测的最大数量。检测过程在找到足够多数量的条码结果后，将结束检测。
 
LPVBarcodeType Type [get, set]
 设置条码检测的类型，可谓多种类型的组合。
 
BOOL VerifyQuality [get, set]
 设置在解码过程中是否进行条码质量检测。默认设置为 False
 

详细描述

该类实现了 ILBarcodeDetector 接口，用于创建条码检测工具的实例。

注：该类在 Linux / MacOS 平台暂不可用。
