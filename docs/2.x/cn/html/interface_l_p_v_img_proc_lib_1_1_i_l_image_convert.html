详细描述

该接口类提供了用于图像类型转换的功能，可进行颜色空间转换、通道操作、位深转换等。

使用这个接口，需要创建一个 LImageConvert 对象。

示例代码

C++

Code snippets: snippets/ColorSpace.cpp

ILImageConvertPtr imgCvter = LImageConvert::Create();
imgCvter->BGRToGray(colorImg, grayImg);
imgCvter->Convert(LPVColorSpaceBGR, LPVColorSpaceHSV, colorImg, hsvImg);
C#

Code snippets: snippets/ColorSpace.cs

LImageConvert imgCvter = new LImageConvert();
imgCvter.BGRToGray(colorImg, grayImg);
imgCvter.Convert(LPVColorSpace.LPVColorSpaceBGR, LPVColorSpace.LPVColorSpaceHSV, colorImg, hsvImg);
LPVCoreLib::LPVColorSpace
LPVColorSpace
该枚举表示图像的颜色空间，同时指定分量通道顺序。所有通道均归一化到 0 ~ 255 范围。
Definition: LPVCore.idl:1665
COM

Code snippets: snippets/ColorSpace.cpp

ILImageConvertPtr imgCvter = LImageConvert::Create();
// or, ILImageConvertPtr imgCvter(__uuidof(LImageConvert)), for absence of built-in Create functions (before 2.2.0)
imgCvter->BGRToGray(colorImg, grayImg);
imgCvter->Convert(LPVColorSpaceBGR, LPVColorSpaceHSV, colorImg, hsvImg);

根据指定的各通道权重，混合 BGR 彩色图像的三个通道

\[ Gray = w0⋅B + w1⋅G + w2⋅R \]

参数
[in]bgrImg 输入的 BGR 彩色图像
[in]w0第一个（蓝色）通道的权重，取值范围在 -255 ~ 255
[in]w1第二个（绿色）通道的权重，取值范围在 -255 ~ 255
[in]w2第三个（红色）通道的权重，取值范围在 -255 ~ 255
[out]grayImg 返回灰度图像，尺寸与输入的彩色图像一致。
