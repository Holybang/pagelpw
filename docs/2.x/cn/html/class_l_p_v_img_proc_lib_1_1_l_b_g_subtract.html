 属性 继承自 ILBGSubtract
double GenerateThreshold [get, set]
 模型生成阈值，用于控制背景模型生成的复杂度。 若前背景图像的差异较大，需要设置一个比较大的生成阈值，获得更为鲁棒的背景模型； 相反若是前背景图像的差异很小，则需要调整为一个比较小的生成阈值，使得背景模型更为敏感。
取值范围为正数，默认设置为 3.
修改该数值将重置背景模型。
 
int HistoryMaxCount [get, set]
 历史数据的最大帧数。这些数据用于构建背景模型。
取值范围为正整数，默认设置为 500.
修改该数值将重置背景模型。
 
double LearningRate [get, set]
 学习率，用于控制背景模型从新一帧图像中学习背景变化的速度。
取值范围为 0 ~ 1，数值越大表示学习速度越快。0 表示不更新背景模型，1 表示按最新一帧图像完全更新背景模型。 默认设置为 -1，算法将在运行过程成基于历史数据和当前模型，自动确定一个合适的学习率。
 
double VarianceThreshold [get, set]
 方差阈值，用于判断当前像素是前景还是背景。当背景变化明显，可将该阈值设高，以降低检测灵敏度，减少误报。
取值范围为正数，默认设置为 25.
修改该数值将重置背景模型。
