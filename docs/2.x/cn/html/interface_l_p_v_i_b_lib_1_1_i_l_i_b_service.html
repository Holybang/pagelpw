详细描述

该接口提供连接到灵闪服务的功能。

灵闪是一个基于利珀科技机器视觉基础框架开发的轻便的智能算法配置及组合工具平台。 通过所见即所得的交互界面和简单的拖拽操作，用于快速开发和迭代定位、测量、缺陷检测等机器视觉任务。 灵闪平台提供一系列覆盖具体应用需求的算法工具包: 特征定位、数量检测、有无判定、基础计算、图像变换、结合测量和计算、相机标定、符号识别、物体分类等等。

LPV IB 服务模块帮助用户在基于灵闪快速编辑机器视觉方案的基础上，将其集成到 LPV 应用程序中运行、操作， 实现更大程度上的灵活性和可扩展性。

使用这个接口，需要创建一个 LIBService 对象。

示例代码

C++
ILIBServicePtr ibService = LIBService::Create();
int errCode = ibService->SetUp(L"path to the IntelliBlink root folder, say D:/Programs/IntelliBlink/");
if (errCode != LPVNoError || !ibService->IsReady()) return; // set up IB service failed
// load task
int taskId = 0;
errCode = ibService->LoadTask(L"mytask.task", &taskId);
if (errCode != LPVNoError) return; // load failed
// load global-actions
errCode = ibService->LoadGlobalAction(L"globalAction.json");
if (errCode != LPVNoError) return; // load failed
// change some of the global variables
ILIBDataPtr newData = LIBData::Create();
newData->FromString(L"this is a test");
ibService->SetGlobalVariableValue(L"test_string", newData);
// execute the task
ILIBTaskResultPtr taskResults;
errCode = ibService->ExecuteTaskSync(taskId, img, &taskResults);
if (errCode != LPVNoError) return; // execution failed
// do other stuff, like parse and display task result, change task configuration...
// see ILIBTaskResult, ILIBTaskConfig for more details
ibService->TearDown();
