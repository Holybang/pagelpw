枚举类型说明

◆ LPVSortedArrayDrawFlags

enum LPVSortedArrayDrawFlags

该枚举用于控制模板匹配结果绘制的行为

枚举值
LPVSortedArrayDrawPosition 

绘制阵列中对象的位置点

LPVSortedArrayDrawIndex 

绘制阵列中对象的索引（x，y）

LPVSortedArrayDrawFiltered 

绘制被过滤的对象的位置点，使用与当前描边笔相反的颜色

LPVSortedArrayDrawMissing 

绘制阵列中缺失的点位置，绘制为 x 符号

LPVSortedArrayDrawRowLines 

绘制行直线，使用与当前描边笔相反的颜色

LPVSortedArrayDrawColumnLines 

绘制列直线，使用与当前描边笔相反的颜色

LPVSortedArrayDrawDefault 

默认值，只绘制阵列中对象的位置点和索引

LPVSortedArrayDrawAll 

绘制所有部分
