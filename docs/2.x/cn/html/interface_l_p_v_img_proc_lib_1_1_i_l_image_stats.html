详细描述

该接口类提供了用于图像的各种属性统计的工具。

使用这个接口，需要创建一个 LImageStats 对象。

示例代码

C++
ILImageStatsPtr imgStats = LImageStats::Create();
int minVal, x, y;
imgStats->Min(img, nullptr, &minVal, &x, &y);
C#
LImageStats imgStats = new LImageStats();
int minVal, x, y;
imgStats.Min(img, null, out minVal, out x, out y);
COM
ILImageStatsPtr imgStats = LImageStats::Create();
// or, ILImageStatsPtr imgStats(__uuidof(LImageStats)), for absence of built-in Create functions (before 2.2.0)
int minVal, x, y;
imgStats->Min(img, NULL, &minVal, &x, &y);

计算输入彩色图像中像素值在设置范围之内、之外的像素总数。
对于每一个像素，若其所有通道的颜色值均在指定范围内，则该像素为范围内“between”像素。 反之，若其任一通道的数值小于或大于指定颜色范围的下限或上限，则该像素为范围外“outside”像素。

参数
[in]img 输入图像。
[in]region输入区域，可为空。
[in]lb颜色范围下限，所有通道的数值需均小于或等于上限
[in]ub颜色范围上限，所有通道的数值需均大于或等于上限
[out]betweenCount输出颜色值在颜色范围之间的像素总数。
[out]outsideCount输出颜色值在颜色范围以外的像素总数。

double Sharpness ( ILImage *  img,

计算输入图像的相对清晰度，基于图像中总体的边缘强度。清晰度计算通常用于实现对静态场景的自动对焦。 注意清晰度通常受到图像的对比度、亮度以及场景内容的影响，因此无法比较不同场景的清晰度数值。

参数
[in]img输入图像。
[in]region输入区域，可为空。
返回值
value返回清晰度评价数值。
