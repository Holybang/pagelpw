int ConnectTolerance [get, set]
 轮廓连接阈值。若两个候选轮廓的端点相近，将连接合并这两个候选轮廓为一个结果。 默认设置为 1 表示不进行额外的轮廓连接。
 
LPVExcludeBoundaryMode ExcludeBoundaryMode [get, set]
 定义如何处理超出图像或检测区域边界的轮廓。仅支持 LPVEBModeEntire 或者 LPVEBModeNone。
 
int FilterShort [get, set]
 相对长度阈值（基于候选轮廓中最大长度的百分比），用于过滤候选轮廓中的短边。 与 FilterWeak 一同使用，用于过滤 短并且弱 的轮廓。
 
int FilterWeak [get, set]
 相对强度阈值（基于候选轮廓中最大强度的百分比），用于过滤候选轮廓中的弱边。 与 FilterShort 一同使用，用于过滤 短并且弱 的轮廓。
 
int KSize [get, set]
 边缘滤波所使用的核尺寸，需要是一个奇数，如 3, 5, 7 等。默认设置为 3 。 对于模糊的图像，可能需要设置更大的核尺寸。
 
int MaxCount [get, set]

int RejectWeak [get, set]
