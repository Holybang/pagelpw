int CountThreshold [get, set]
 用于拟合的最少点数，若点数少于该数值，将终止拟合迭代，拟合失败。 默认值为 0 表示不进行最少点数检查。
 
double DistThreshold [get, set]
 拟合图形与点集中的用于拟合的点之间的距离阈值。拟合迭代过程中，距离大于该阈值的点将被剔除不参与拟合。 默认值为 0 表示跳过剔除过程，即所有点总是都参与拟合。
 
int MaxIteration [get, set]
 迭代拟合的最大次数。默认值为 10.
 

详细描述

该接口类提供了多种图形拟合算法，用于基于给定点集拟合几何图形。

相比于 ILPoints 接口中所提供的拟合函数， 该接口能够剔除输入中的异常点，使得拟合结果更稳定，且易于处理输入点不足的情况。

使用这个接口，需要创建一个 LFitting 对象。

示例代码

C++
ILFittingPtr fittingTool = LFitting::Create();
fittingTool->DistThreshold = 3;
fittingTool->CountThreshold = 10;
ILLinePtr line = fittingTool->FitLine(points);
C#
LFitting fittingTool = new LFitting();
fittingTool.DistThreshold = 3;
fittingTool.CountThreshold = 10;
LLine line = fittingTool.FitLine(points);
COM
ILFittingPtr fittingTool = LFitting::Create();
// or, ILFittingPtr fittingTool(__uuidof(LFitting)), for absence of built-in Create functions (before 2.2.0)
fittingTool->DistThreshold = 3;
fittingTool->CountThreshold = 10;
ILLinePtr line = fittingTool->FitLine(points);
