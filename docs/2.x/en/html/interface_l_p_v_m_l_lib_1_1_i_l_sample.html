Detailed Description

This interface represents a sample instance, which usually contains an image and an optional region object.

The samples are usually organized and managed in the sample database LSampleDatabase. To use this interface, you should create a LSample object or fetch it from the database.

Example Code

C++
// create the sample then add to the database
ILSamplePtr sample1 = LSample::Create();
sample1->Set(img, region);
sampleDb->AddSample(LPVSampleType::LPVSampleTrain, sample1);
// create then get back from the database
std::wstring sampleName = sampleDb->AddSampleImage(LPVSampleType::LPVSampleTrain, img, region);
ILSamplePtr sample2 = db2->GetSample(sampleName);
