Detailed Description

This interface provide functionalities of the wafer map display control.

Usually you add the display control via the Toolbox in Visual Studio. This interface helps you to manage the scene in the display control, for the image, regions, geometric shapes and algorithms' results to shown in the control.

See ILWaferMapDisplayEvents for the control's event.

Example Code

C++
// lpvWaferDisplay is a COM ActiveX library, it supports only COM interface.
// for every pass-in LPV/LPW object, you should cast it to IDispatch via LObjectToDispatch() function.
// since 2.2.0, use the built-in implicit type casting.
// set wafer map to the control
displayCtrl->SetWaferMap(LObjectToDispatch(*waferMap));
displayCtrl->SetWaferMap(*waferMap); // since 2.2.0
// get wafer map position from client position (top-left corner of the window)
double x, y;
displayCtrl->ClientToWafer(0, 0, &x, &y);
C#
// set wafer map to the control
displayCtrl.SetWaferMap(waferMap);
// get wafer map position from client position (top-left corner of the window)
double x, y;
displayCtrl.ClientToWafer(0, 0, out x, out y);
COM
// set wafer map to the control
displayCtrl->SetWaferMap(waferMap);
// get wafer map position from client position (top-left corner of the window)
double x, y;
displayCtrl->ClientToWafer(0, 0, &x, &y);

Note: This interface is currently not available on Linux / MacOS platform.
