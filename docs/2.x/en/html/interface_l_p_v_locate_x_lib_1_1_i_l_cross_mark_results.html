Detailed Description

This interface holds a collection of cross mark detection results.

Usually you get the LCrossMarkResults object from the ILCrossMarkDetector::Detect() as detection result. You can then use this interface to access individual result and draw the results on graphics windows.

Example Code

C++
ILCrossMarkResultsPtr cmResults;
LPVErrorCode err = cmDetector->Detect(img, roi, &cmResults);
if (err != LPVErrorCode::LPVNoError) return;
int c = cmResults->Count();
for (int i = 0; i < c; ++i) {
ILCrossMarkResultPtr result = cmResults->Item(i);
// handle each cross mark result
}
