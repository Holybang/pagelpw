Detailed Description

This interface provide functionalities of the display control.

Usually you add the display control via the Toolbox in Visual Studio. This interface helps you to manage the scene in the display control, for the image, regions, geometric shapes and algorithms' results to shown in the control.

See ILDisplayEvents for the control's event.

Example Code

C++
// lpvDisplay is a COM ActiveX library, it supports only COM interface.
// for every pass-in LPV object, you should cast it to IDispatch via LObjectToDispatch() function.
// since 2.2.0, use the built-in implicit type casting.
// set image to the control
displayCtrl->SetImage(LObjectToDispatch(*img));
displayCtrl->SetImage(*img); // since 2.2.0
// add a rectangle region to the control
ILRectRegionPtr roi = LRectRegion::Create();
roi->SetPlacement(10, 10, 200, 100);
displayCtrl->AddRegion(LObjectToDispatch(*roi), true); // it's interactive
displayCtrl->AddRegion(*roi, true); // since 2.2.0
// add a circle to the control
ILCirclePtr circle = LCircle::Create();
circle->Set(100, 50, 50);
displayCtrl->AddObject(LObjectToDispatch(*circle), 0); // use default draw flags
displayCtrl->AddObject(*circle, 0); // since 2.2.0
// add the pattern matching result to the control
displayCtrl->AddObject(LObjectToDispatch(*matchResults),
(LPVPatDrawFlags)(LPVPatDrawFlags::LPVPatDrawBoundingRect | LPVPatDrawFlags::LPVPatDrawCenter));
displayCtrl->AddObject(*matchResults, LPVPatDrawFlags::LPVPatDrawDefault); // since 2.2.0
