int GlueEdgeThreshold [get, set]
 The glue edge threshold used to determine the glue range and help the noise removal in edge detection. It's a relative threshold in 0 ~ 100, higher value results in the smaller glue range. The working threshold is computed on-the-fly by the specified percentage and the average luminance of the detection region. By default, it's set to 100, that in fact glue range detection is not enabled.
