Detailed Description

This interface holds a single kerf detection result and is used to access its properties.

Usually you get the LKerfResult object from the ILKerfDetector::Detect1() or ILKerfDetector::Detect2() function. You can then use this interface to access to the detailed informations and draw the result on graphics windows.

Example Code

C++
ILKerfResultPtr result;
LPVErrorCode err = kerfTool->Detect1(img, rotRectRoi, &result);
if (err != LPVNoError) return; // no result or failed
double kerfWidth = result->LineUp->Distance(result->LineDown);
