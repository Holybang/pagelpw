Detailed Description

This interface provide functionalities of edge extraction.

To use this interface, you should create a LEdgeFilter object.

Example Code

C++
ILEdgeFilterPtr edgeOp = LEdgeFilter::Create();
edgeOp->SetEdgeKernel(LPVSobel, 5, 5);
edgeOp->GradientFull(img, resultImg);
C#
LEdgeFilter edgeOp = new LEdgeFilter();
edgeOp.SetEdgeKernel(LPVEdgeKernel.LPVSobel, 5, 5);
edgeOp.GradientFull(img, resultImg);
