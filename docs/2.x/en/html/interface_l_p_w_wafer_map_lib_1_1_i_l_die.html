Detailed Description

This interface holds a single die and is used to access its properties.

To use this interface, you should create a LDie object, or get the LDie object from the wafer map.

Example Code

C++
ILDiePtr die = wafermap->GetDie(colIdx, rowIdx);
die->Quality = LPWBinQuality::LPWBinPass; // mark the die as pass
double x = die->X; // get its x-coordinate
C#
LDie die = wafermap.Die[colIdx][rowIdx];
die.Quality = LPWBinQuality.LPWBinPass; // mark the die as pass
double x = die.X; // get its x-coordinate
