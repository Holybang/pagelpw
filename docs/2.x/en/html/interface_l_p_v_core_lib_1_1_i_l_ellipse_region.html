Detailed Description

This interface expose APIs to create/modify an ellipse region object.

To use this interface, you should create a LEllipseRegion object.

Example Code

C++
ILEllipseRegionPtr ellipseRoi = LEllipseRegion::Create();
annulusRoi->SetPlacement(200, 100, 50, 100, 15);
C#
LEllipseRegion annulusRoi = new LEllipseRegion();
annulusRoi.SetPlacement(200, 100, 50, 100, 15);
COM
ILEllipseRegionPtr ellipseRoi = LEllipseRegion::Create();
// or, ILEllipseRegionPtr ellipseRoi(__uuidof(LEllipseRegion)), for absence of built-in Create functions (before 2.2.0)
annulusRoi->SetPlacement(200, 100, 50, 100, 15);
