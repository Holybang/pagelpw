Detailed Description

This interface represents a sample database for training and testing sample organization and management.

There are two supported database type:

  • Persisted: The sample files and database information is serialized to the disk in the working directory, when the database is changed, for example, any sample are added or removed.
  • In-memory: The database relies primarily on the main memory for data storage, which means all the sample files and informations are never saved to the disk and will be lost when the application restarted.

To use this interface, you should create a LSampleDatabase object.

Example Code

C++
// create "mylabel" sample database
ILSampleDatabasePtr sampleDb(__uuidof(LSampleDatabase));
sampleDb->Create(L"mylabel", L"d:/test/mylabel");
// add sample images
sampleDb->AddSampleImage(LPVSampleType::LPVSampleTrain, img1, region1);
sampleDb->AddSampleImage(LPVSampleType::LPVSampleTest, img2, region2);
// load "anotherlabel" sample database
ILSampleDatabasePtr sampleDb2(__uuidof(LSampleDatabase));
sampleDb2->Load(L"d:/test/anotherlabel");
C#
// create "mylabel" sample database
LSampleDatabase sampleDb = new LSampleDatabase();
sampleDb.Create("mylabel", "d:/test/mylabel");
// add sample images
sampleDb.AddSampleImage(LPVSampleType.LPVSampleTrain, img1, region1);
sampleDb.AddSampleImage(LPVSampleType.LPVSampleTest, img2, region2);
// load "anotherlabel" sample database
LSampleDatabase sampleDb2 = new LSampleDatabase();
sampleDb2.Load("d:/test/anotherlabel");

Create and initialized the database.

Parameters
[in]labelNameThe label of the sample database.
[in]workingDirectoryThe working directory. To create an in-memory database, pass in an empty path. ◆ Refresh()

LPVErrorCode Refresh ( )

Refresh the dataset manually. For the in-memory dataset, it makes no sense. For the persisted, all sample folders are re-scanned and all samples are re-loaded. Inaccessible folders and files are ignored and skipped.
It is time-consuming, you may not do this frequently.

Return values
errorReturn error code if anything is wrong.
