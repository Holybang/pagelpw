Detailed Description

This interface expose APIs to create/modify/manipulate/draw a LPV region object (LRegion , LRectRegion , LRotRectRegion , LCircleRegion , etc.).

Region object is useful to restrict the detection and inspection to required region.
To use this interface, you should create a region object.

There are several ways to modify a region's placement. The most popular options are listed below:

  1. For simple geometric region, do it programmatically.
  2. Interactive with the region by dragging the handles.
  3. Create a new region via logical operators on multiple existing regions.

Example Code

C++
// 1. set placement programmatically
rect_roi->SetPlacement(10, 10, 200, 100);
// 2. drag the region
move_handle = roi->HitTest(cursor_x, cursor_y, zoomx, zoomy);
roi->Drag(move_handle, cursor_x, cursor_y, zoomx, zoomy);
// 3. calculated using existing regions
ILRegionPtr new_roi = rect_roi->Add(rect_roi_other);
// 4. map the position of a line from the image to the region
ILLinePtr line_in_roi = rect_roi->MapObjectFromImage(line);
// 5. map the position of the blob results from the region back to the image
ILBlobResultsPtr blobs_in_img = rect_roi->MapObjectToImage(blob_in_roi);
C#
// 1. set placement programmatically
rect_roi.SetPlacement(10, 10, 200, 100);
// 2. drag the region
move_handle = roi.HitTest(cursor_x, cursor_y, zoomx, zoomy);
roi.Drag(move_handle, cursor_x, cursor_y, zoomx, zoomy);
// 3. calculated using existing regions
LRegion new_roi = rect_roi.Add(rect_roi_other);
// 4. map the position of a line from the image to the region
LLine line_in_roi = rect_roi.MapObjectFromImage(line);
// 5. map the position of the blob results from the region back to the image
LBlobResults blobs_in_img = rect_roi.MapObjectToImage(blob_in_roi);
