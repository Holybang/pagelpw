Detailed Description

This interface holds the device information of the wafer.

To use this interface, you should create a LWaferProps object, or get the LWaferProps object from the wafer map.

Example Code

C++
ILWaferPropsPtr props(__uuidof(LWaferProps));
props->DeviceID = L"My Wafer";
props->Diameter = 200;
props->DieSizeX = 2.25;
props->DieSizeY = 1.35;
// synthesize the wafer map using the properties
ILWaferMapPtr wmap(__uuidof(LWaferMap));
wmap->Synthesize(props);
C#
LWaferProps props = new LWaferProps();
props.DeviceID = "My Wafer";
props.Diameter = 200;
props.DieSizeX = 2.25;
props.DieSizeY = 1.35;
// synthesize the wafer map using the properties
LWaferMap wmap = new LWaferMap();
wmap.Synthesize(props);
