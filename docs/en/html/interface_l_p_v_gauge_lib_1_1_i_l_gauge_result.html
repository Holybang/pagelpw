Detailed Description

This interface holds a gauging result and is used to access its properties.

Usually you get the LGaugeResult object from the gauge like LLineGauge for line shape inspection. You can then use this interface to access to the detection result and details, and draw the result on graphics windows.

Example Code

C++
ILLinePtr line = lineGaugeResult->Result;
ILPointsPtr allPoints = lineGaugeResult->SamplePoints;
C#
LLine line = lineGaugeResult.Result;
LPoints allPoints = lineGaugeResult.SamplePoints;
