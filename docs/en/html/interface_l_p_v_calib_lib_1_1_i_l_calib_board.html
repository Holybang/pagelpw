Detailed Description

This interface provide functionalities of the calibration board.

The calibration board is used to detect and generate image and world point pairs for calibration.
We support 3 types of board, see LPVCalibBoardType for more details.

To use this interface, you should create a LCalibBoard object.

Example Code

C++
ILCalibBoardPtr m_calib_board(__uuidof(LCalibBoard));
m_calib_board->GridWidth = 10;
m_calib_board->GridHeight = 7;
m_calib_board->SquareSize = 1;
LPVErrorCode err = m_calib_board->Detect(m_img, &m_pnt_pairs);
if (err != LPVErrorCode::LPVNoError) return;
