Detailed Description

This interface represent the wafer map, provide functionalities of the wafer map synthesize, serialization, display, die accessing and counting.

Wafer maps are used in the backend of the IC manufacturing process to annotate the die on a wafer. The LWaferMap object is designed for the wafer map viewing/editing/analysis, which can be used by the equipment OEMs to work with probers and testers to display, update and output map files.

To use this interface, you should create a LWaferMap object.

Example Code

C++
ILWaferMapPtr templateMap(__uuidof(LWaferMap));
templateMap->Load(L"template.sinf");
// move the center of the template map to (0, 0)
templateMap->WaferProps->CenterX = 0;
templateMap->WaferProps->CenterY = 0;
ILWaferSorterPtr sorter(__uuidof(LWaferSorter));
// set the template
sorter->Template = templateMap;
// add dice
// sort
ILWaferMapPtr wmap;
LPVErrorCode err = sorter->Sort(&wmap);
if (err != LPVErrorCode.LPVNoError) return;
// count the pass dice
int passCount = wmap->CountByQuality(LPWBinPass);
int totalCount = wmap->Count();
double passPercentage = (double)passCount / totalCount;

◆ Synthesize()

LPVErrorCode Synthesize ( ILWaferProps *  prop)

Synthesize the wafer map using the given properties.
All positions that entirely outside the wafer are generated as Null die, those completely within the wafer and the margin) are generated as Pass die, the others that touches or lies outside the margin are Edge die.
You may then edit the wafer map later via Die or DieQuality to add reference/skip or other kinds of dice.

Parameters
[in]propThe input wafer properties. The step size and the column/row count(or diameter) is must-have.
Return values
errorReturn error code if anything is wrong.
