Detailed Description

This interface expose APIs to change graphics settings of a drawable class.

All other LPV classes that provides draw functionalities implements this interface. To use this interface, you could either cast the LPV classes to ILDrawable interface, or call ILSystem to do it for you. See ILSystem for more information.

The following lists all the drawable LPV classes:

LPVCoreLibLRegion, LRectRegion, LRotRectRegion, LCircleRegion, LPolyRegion, LMaskRegion, LAnnulusRegion, LEllipseRegion, LAnnulusSectorRegion
LPVGeomLibLPoint, LPoints, LCircle, LLine, LPolygon, LRect, LEllipse, LArc
LPVGeomXLibLSortedArray
LPVPatLibLMatchResult, LMatchResults
LPVCalibLibLCalibPointPairs
LPVImgProcLibLHistogram
LPVBlobLibLBlob, LBlobResults
LPVLocateLibLLineResult, LLineResults, LCircleResult, LCircleResults

Example Code

C++
((ILDrawablePtr)drawable)->SetPenWidth(2);
// or do it via LSystem object
sys->SetPen(roi, LPVPenStyle::LPVPenSolid, 1, LPVColor::LPVYellow);
C#
((ILDrawable)roi).SetPenWidth(2);
// or do it via LSystem object
sys.SetPen(roi, LPVPenStyle.LPVPenSolid, 1, LPVColor.LPVYellow);
