Detailed Description

This interface provide functionalities of circle inspection via the gauge.

The circle gauge is used to measure position and size of the circle formed by the circular edges. Edge points are sampled along the given region using regularly spaced point measurement gauges, and then fit to a circle.

Compared with the feature locating tool LCircleDetector, it's more resistant to noise and efficient. But it is easily confused when there's multiple concentric circles in the region. For that case and for high-accuracy required task, it's recommended to use LCircleDetector instead.

To use this interface, you should create a LCircleGauge object.

Example Code

C++
ILCircleGaugePtr circleGauge(__uuidof(LCircleGauge));
circleGauge->Polarity = LPVPolarity::LPVBlack2White;
circleGauge->AcceptScore = 40;
ILGaugeResultPtr circleGaugeResult;
LPVErrorCode err = circleGauge->Detect(img, annulus_roi, &circleGaugeResult);
if (err != LPVErrorCode::LPVNoError) return;
