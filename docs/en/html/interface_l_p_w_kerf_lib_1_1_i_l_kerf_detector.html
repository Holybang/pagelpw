int GlueEdgeThreshold [get, set]
 The glue edge threshold used to determine the glue range and help the noise removal in edge detection. It's a relative threshold in 0 ~ 100, higher value results in the smaller glue range. The working threshold is computed on-the-fly by the specified percentage and the average luminance of the detection region. By default, it's set to 100, that in fact glue range detection is not enabled.

Detailed Description

This interface provide functionalities of cutting kerf detection.

Kerf detector is designed to locate a single(wide) or double(narrow) cutting kerf edges in given image and along the region's direction.
The algorithm is reliable and resistant to noise, which may be the glue on the cutting or the various and unpredictable background.

To use this interface, you should create a LKerfDetector object.

Example Code

C++
ILKerfDetectorPtr m_kerftool(__uuidof(LKerfDetector));
m_kerftool->KerfWidth = 10;
m_kerftool->KerfDistance = 40;
m_kerftool->ContrastThreshold = 2;
ILKerfResultPtr result1, result2;
LPVErrorCode err = m_kerftool->Detect2(m_img, m_rotroi, &result1, &result2);
if (err != LPVNoError) return; // no result or failed
