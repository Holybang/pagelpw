Detailed Description

This interface provide functionalities of edge extraction.

To use this interface, you should create a LEdgeFilter object.

Example Code

C++
ILEdgeFilterPtr m_edge_filter_op(__uuidof(LEdgeFilter));
m_edge_filter_op->SetEdgeKernel(LPVSobel, 5, 5);
m_edge_filter_op->GradientFull(m_img, m_result_img);

Set the kernel used in edge filtering of specified type and size.

Parameters
[in]kTypeThe kernel Type.
[in]kWidthThe width of the kernel size, should be an odd number, say, 3, 5, 7, etc. If a kernel size is set to be 1, the kernel degenerated to be \( (-1, 1) \) no mater what kernel type it is.
[in]kHeightThe height of the kernel size, should be an odd number, say, 3, 5, 7, etc.
