Detailed Description

This interface holds a collection of data blob results.

Usually you get the LDataBlobResults object from the ILDataBlobAnalysis::Build() as analysis result. You can then use this interface to access individual result.

Example Code

C++
ILDataBlobResultsPtr m_data_blob_results;
LPVErrorCode err = m_data_blob_ana->Build(m_data_list, &m_data_blob_results);
if (err != LPVErrorCode::LPVNoError) return;
int c = m_data_blob_results->Count();
for (int i = 0; i < c; ++i) {
ILDataBlobPtr blob = m_data_blob_results->Item(i);
// handle each blob result
}
